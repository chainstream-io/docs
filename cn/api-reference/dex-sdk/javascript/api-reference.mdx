本文档描述了 ChainStream DEX JavaScript SDK 提供的方法。每个方法都是对应 REST API 的封装，详细的参数说明和返回值结构请点击对应的 API 链接查看。

## DEX 操作

### 获取 DEX 列表
获取指定链上的所有DEX列表。

<Tip>
  **REST API:** [DEX - 列表](/cn/api-reference/endpoint/dex/dex)
</Tip>

```typescript
client.dex.list()
```

### 代币铸造
在指定DEX上创建新代币。

<Tip>
  **REST API:** [DEX - 铸造](/cn/api-reference/endpoint/dex/dex-chain-mint)
</Tip>

```typescript
client.dex.mint()
```

### 兑换
兑换代币。

<Tip>
  **REST API:** [DEX - 兑换](/cn/api-reference/endpoint/dex/dex-chain-swap)
</Tip>

```typescript
client.dex.swap()
```

### 获取兑换路由
计算代币兑换的最佳路由。

<Tip>
  **REST API:** [DEX - 路由](/cn/api-reference/endpoint/dex/dex-chain-route)
</Tip>

```typescript

```
