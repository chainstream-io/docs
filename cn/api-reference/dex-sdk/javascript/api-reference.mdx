# API 参考

## DEX 操作

### 获取 DEX 列表
获取指定链上的所有DEX列表。

```typescript
client.dex.list(params: {
  chain: string;  // 链名称
}): Promise<DexInfo[]>

interface DexInfo {
  program_address: string;  // DEX程序地址
  protocol_family: string;  // 协议族
  protocol_name: string;    // 协议名称
  chain: string;           // 链
}
```

### 代币铸造
在指定DEX上创建新代币。

```typescript
client.dex.mint(params: {
  chain: string;           // 链名称
  dex: string;            // DEX名称
  name: string;           // 代币名称
  symbol: string;         // 代币符号
  uri?: string;           // 元数据URI
  description?: string;   // 代币描述
  migration_dex?: string; // 迁移目标DEX
  icon?: string;         // 代币图标
  banner?: string;       // 横幅图片
  token_amount?: string; // 代币总供应量
}): Promise<MintResponse>

interface MintResponse {
  serialized_tx: string;   // Base64编码的交易
  mint_address: string;    // 代币铸造地址
  elapsed_time: number;    // 处理时间（毫秒）
  extra: any;             // 额外元数据
}
```

### 提交铸造交易
提交已签名的代币铸造交易。

```typescript
client.dex.submitMint(params: {
  chain: string;          // 链名称
  dex: string;           // DEX名称
  signed_tx: string;     // 已签名的交易
}): Promise<SubmitMintResponse>

interface SubmitMintResponse {
  signature: string;      // 交易签名
  elapsed_time: number;   // 处理时间
}
```

### 获取兑换路由
计算代币兑换的最佳路由。

```typescript
client.dex.route(params: {
  chain: string;          // 链名称
  dex: string;           // DEX名称
  input_mint: string;     // 输入代币地址
  output_mint: string;    // 输出代币地址
  amount: string;         // 数量
  mode?: 'ExactIn' | 'ExactOut';  // 兑换模式
  slippage?: string;     // 滑点
}): Promise<RouteResponse>

interface RouteResponse {
  serialized_tx: string;  // Base64编码的交易
  elapsed_time: number;   // 处理时间
  route_info: {
    input_amount: string;
    output_amount: string;
    price_impact: string;
    fee: string;
  };
}
```

### 执行兑换
执行代币兑换交易。

```typescript
client.dex.swap(params: {
  chain: string;          // 链名称
  dex: string;           // DEX名称
  input_mint: string;     // 输入代币地址
  output_mint: string;    // 输出代币地址
  amount: string;         // 数量
  mode?: 'ExactIn' | 'ExactOut';  // 兑换模式
  slippage?: string;     // 滑点
  pool_address?: string; // 资金池地址
}): Promise<SwapResponse>

interface SwapResponse {
  serialized_tx: string;  // Base64编码的交易
  elapsed_time: number;   // 处理时间
  args: SwapArgs;        // 兑换参数
  route_info: RouteInfo; // 路由信息
}
```

## 代币操作

### 获取代币信息
获取代币的详细信息。

```typescript
client.token.get(params: {
  chain: string;         // 链名称
  tokenAddress: string;  // 代币地址
}): Promise<TokenInfo>

interface TokenInfo {
  metadata: {
    chain: string;
    decimals: number;
    name: string;
    symbol: string;
    metadata_address: string;
    address: string;
    token_creators: string[];
    image_url: string;
    uri: string;
    description: string;
  };
  price: {
    value: string;
    updated_at: number;
  };
  market_cap: {
    circulating_supply: string;
    market_cap: string;
    holders: number;
  };
}
```

## 订单操作

### 下单
创建新的订单。

```typescript
client.order.place(params: {
  chain: string;           // 链名称
  dex: string;            // DEX名称
  market_address: string;  // 市场地址
  side: 'buy' | 'sell';   // 订单方向
  amount: string;         // 数量
  price: string;          // 价格
  priority_fee?: string;  // 优先费用
}): Promise<OrderResponse>

interface OrderResponse {
  serialized_tx: string;
  extra: any;
  order_id: string;
  status: string;
  filled_amount: string;
}
```

### 取消订单
取消已存在的订单。

```typescript
client.order.cancel(params: {
  chain: string;           // 链名称
  dex: string;            // DEX名称
  market_address: string;  // 市场地址
  order_id: string;       // 订单ID
}): Promise<void>
```

### 取消全部订单
取消指定市场的所有订单。

```typescript
client.order.cancelAll(params: {
  chain: string;           // 链名称
  dex: string;            // DEX名称
  market_address: string;  // 市场地址

```

### 吃单


## 交易操作

### 发送交易
提交已签名的交易到区块链。

```typescript
client.transaction.send(params: {
  chain: string;          // 链名称
  signed_tx: string;      // Base64编码的已签名交易
  submit_type?: string;   // 提交类型
}): Promise<TransactionResponse>

interface TransactionResponse {
  signature: string;      // 交易签名/哈希
  elapsed_time: number;   // 处理时间（毫秒）
}
```

### 获取交易详情
获取交易的详细信息。

```typescript
client.trade.get(params: {

}): Promise<TradeDetail>

interface TradeDetail {
  tx: string;
  amount: string;
  price_usd: string;
  volume: string;
  type: string;
  wallet: string;
  time: number;
  program: string;
}
```

### 获取代币交易
