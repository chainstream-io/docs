## 概述

ChainStream DEX WebSocket API 提供实时数据订阅服务，支持以下数据类型：
- K线数据 (Candles)
- 代币统计 (Token Stats)
- 代币活动 (Token Activities)
- 交易事件 (Trade Events)
- 代币持有者 (Token Holders)
- 钱包余额 (Wallet Balance)
- 代币余额 (Token Balance)
- 代币市值 (Market Cap)
- 代币PNL (Profit and Loss)

**基础 URL**
```bash
wss://realtime-dex.chainstream.io/connection/websocket
```

ChainStream DEX SDK 提供了实时数据订阅功能，支持以下数据类型：
- K线数据
- 代币统计
- 交易活动
- 钱包余额

## 订阅方法

### K线数据订阅

```typescript
const unsubscribe = client.stream.subscribeTokenCandles({
  chain: string;
  tokenAddress: string;
  resolution: '1m' | '5m' | '15m' | '1h' | '4h' | '1d';
  callback: (data: Candle) => void;
}): () => void

interface Candle {
  o: number;    // 开盘价
  c: number;    // 收盘价
  h: number;    // 最高价
  l: number;    // 最低价
  v: number;    // 成交量
  r: string;    // 周期
  t: number;    // 时间戳
}
```

## 数据订阅

### K线数据
实时获取代币价格走势的K线图数据。

**订阅格式**
\`dex-candle:${chain}_${tokenAddress}_${resolution}\`

**参数说明**
| 参数 | 说明 | 示例 |
|------|------|------|
| chain | 区块链名称 | solana |
| tokenAddress | 代币合约地址 | 6p6xgHyF7AeE6TZkSmFsko444wqoP15icUSqi2jfGiPN |
| resolution | K线周期 | 1m, 5m, 15m, 1h, 4h, 1d |

**响应数据**
```typescript
{
  "o": number,    // 开盘价 (open)
  "c": number,    // 收盘价 (close)
  "h": number,    // 最高价 (high)
  "l": number,    // 最低价 (low)
  "v": number,    // 成交量 (volume)
  "r": string,    // 周期 (resolution)
  "t": number     // 时间戳 (timestamp)
}
```

### 代币统计

## 使用示例

## 错误处理

### 错误代码

| 代码 | 描述 | 处理建议 |
|------|------|---------|
| 1000 | 正常关闭 | 无需处理 |
| 1006 | 异常关闭 | 触发重连 |
| 4001 | 认证失败 | 检查令牌 |
| 4004 | 订阅失败 | 检查参数 |

### 重连策略

推荐使用指数退避算法进行重连：

```typescript
function reconnect(attempt) {
  const delay = Math.min(1000 * Math.pow(2, attempt), 10000);
  setTimeout(() => {
    connect();
  }, delay);
}
```

## 使用限制

| 项目 | 限制 | 说明 |
|------|------|------|
| 最大订阅数 | 每个连接100个 | 超出将被拒绝 |
| 消息大小 | 100KB | 超出将被截断 |
| 心跳间隔 | 30秒 | 需要定期发送心跳 |

2. **错误处理**
