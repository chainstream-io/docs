本文档描述了 ChainStream DEX JavaScript SDK 提供的 WebSocket 订阅方法。每个方法都是对应 WebSocket API 的封装，详细的参数说明和返回值结构请点击对应的 API 链接查看。

## K线数据订阅

### 订阅K线数据
订阅代币价格K线数据。

<Tip>
  **WebSocket API:** [K线数据](/cn/api-reference/endpoint/websocket/api#k线数据)
</Tip>

```typescript
client.stream.subscribeTokenCandles()
```

## 连接管理

### 建立连接
```typescript
client.stream.connect()
```

所有订阅方法都会返回一个 `Unsubscrible` 对象，可以用于取消订阅：

```typescript
const subscription = client.stream.subscribeTokenCandles(/* ... */);
// 取消订阅
subscription.unsubscribe();
```

详细的参数说明、数据格式和错误处理请参考 [WebSocket API 文档](/cn/api-reference/endpoint/websocket/api)。
