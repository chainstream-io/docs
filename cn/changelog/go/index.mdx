本文档记录了 ChainStream Go SDK 的所有重要更新、新功能和错误修复。

## September 28,2025 - v0.1.7

#### 🚀 新功能

**Go SDK - 初始发布**
- 发布 Go SDK v0.1.7，提供完整的 API 支持
- 可通过 `go get github.com/chainstream-io/chainstream-go-sdk@v0.1.7` 安装
- 新增 Go SDK 的完整文档
- 新增 Go 开发者快速开始指南
- 新增所有端点的 API 参考文档
- 新增实时数据的 WebSocket 订阅支持
- 新增所有主要 API 端点支持，包括代币、钱包、交易、DEX、排行榜等
- 新增 Webhook 端点管理功能
- 新增 DeFi 协议支持（Moonshot、PumpFun）
- 新增红包功能
- 新增 IPFS 集成
- 新增区块链数据访问
