本文档记录了 ChainStream API 和 SDK 的所有重要更新、新功能和错误修复。

## September 10,2025 - v0.0.9

#### 🔧 功能改进

**Token 接口字段重命名**
- `extra.outerProgramAddress` 重命名为 `extra.launchFromProgramAddress`
- `extra.outerProgramProtocolFamily` 重命名为 `extra.launchFromProgramProtocolFamily`
- `extra.migratedProgramAddress` 重命名为 `extra.migratedToProgramAddress`
- `extra.migratedProtocolFamily` 重命名为 `extra.migratedToProtocolFamily`
- 受影响的接口：Token - Metadata、Token - Metadata (Multi)、Token - Detail、Token - Detail (Multi)

**WebSocket API 更新**
- 更新 Ranking Token List 接口的返回结构
- 新增 `decimals` 字段到 TokenMetadata
- 新增 `launchFrom` 和 `migratedTo` 字段到 TokenMetadata
- 新增 `bondingCurve` 绑定曲线信息
- 完善所有时间段的统计数据（5m, 15m, 30m, 1h, 4h, 24h）
- 更新 SDK 返回结构为嵌套格式
- 新增 Token Holders Statistics 接口的创建者相关字段
- 新增 `creatorsHolders`、`creatorsAmount`、`creatorsRatio` 字段

## September 3,2025 - v0.0.4

#### 🚀 新功能

**Token - Metadata 接口类型优化**
  - `decimals` 字段类型由 `string` 更新为 `number`
  - `tokenCreators` 字段类型由 `jsonstring` 更新为 `object`
  - `extra` 字段类型更新为 `object`
  - `socialMedias` 字段类型由 `jsonstring` 更新为 `object`
  - 具体对象属性请参考相关文档
  - 受影响的接口还包括：Token - Metadata(Multi)、Token - Detail、Token - Detail(Multi)

**Token - Market Data 接口增强**
- 新增 `completionRatio` 毕业完成进度
- 新增 `creatorsCount` 创建者数量
- 新增 `creatorHoldings` 创建者持有量
- 新增 `creatorHoldingRatio` 创建者持有比例

**Ranking 系列接口增强**
- 新增 `tokenCreators` 创建者信息
- 新增 `buys` 某时段买入数
- 新增 `sells` 某时段卖出数

**新增接口**
- **Token - List(Filtered)**：可根据多种条件过滤 tokenList
- **Ranking - FinalStretch Tokens**：获取即将毕业代币列表接口，支持 tag 字段筛选不同协议
- **Ranking - Migrated Tokens**：获取已毕业代币列表接口，支持 tag 字段筛选不同协议

**WebSocket 订阅增强**
- 新增 `subscribeRankingTokensList` 订阅
- 新增 `subscribeRankingTokensStats` 订阅
- 新增 `subscribeRankingTokensHolders` 订阅
- 新增 `subscribeRankingTokensSupply` 订阅
- 新增 `subscribeRankingTokensLiquidity` 订阅
- 新增 `subscribeTokenTrade` 订阅
- 新增 `subscribeWalletPnlList` 订阅
- 新增 `subscribeDexPoolBalance` 订阅
- 新增 `batchSubscribe` 批量订阅方法，支持相同/不同的订阅方法同时批量订阅

#### 🔧 功能改进

**WebSocket 订阅优化**
- 所有 WebSocket 订阅方法中新增了 `filter` 字段
- 支持使用 Google Common Expression Language (CEL) 表达式语言进行筛选过滤

#### 🐛 错误修复

- 修复 Token - Liquidity 接口查询异常 bug
