**Tokens** 数据产品是 ChainStream 索引的所有同质化代币的规范来源。一套 schema、一套地址空间（按链区分）、一套字段——不论你查的是蓝筹 ERC-20、新发 Solana meme 币，还是 BSC 上的交易对。

## 包含什么

| 字段组 | 举例 |
|:--|:--|
| 身份 | `chainId`, `address`, `name`, `symbol`, `decimals`, `logo` |
| 行情 | `priceUsd`, `priceChange24h`, `marketCap`, `fdv`, `volume24h` |
| 供给 | `totalSupply`, `circulatingSupply`, `holderCount` |
| 创建 | `createdAtBlock`, `creatorAddress`, `firstTxHash`, `launchpad` |
| 安全 | `isHoneypot`, `canBuy`, `canSell`, `taxBuy`, `taxSell`, `owner` |
| 社交 | `website`, `twitter`, `telegram`, `verifiedSources` |

安全与社交字段为尽力而为，具体置信度见 [数据新鲜度](/cn/docs/data-products/data-freshness)。

## 常见用例

- 把 `PEPE` 这样的符号解析到用户真正想要的那条链上的正确地址
- 渲染带价格、供给、安全分数、社交信息的代币页
- 用创建时间过滤器发现新发代币
- 按成交量、市值、持币人增速排序
- 给 LLM 喂一个代币"快照"让它做分析

## 计费

REST / GraphQL 每次调用 **1 个 read unit**；安全与完整元数据端点可能更贵。WebSocket 与 Kafka 按投递条数计费。见 [计费与 unit](/cn/docs/platform/billing-payments/billing-and-units)。
