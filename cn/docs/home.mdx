## ChainStream 能给你什么

### 统一的多链数据

覆盖全部支持链的单一 schema。代币价格、DEX 交易、钱包 PnL、池子流动性、持币人分布、OHLC K 线、Smart Money 活动、区块链底层数据——全部通过同一套鉴权、同一种 ID、同一组维度过滤器访问。

### 匹配你技术栈的实时投递

<CardGroup cols={3}>
  <Card title="REST + GraphQL" icon="bolt">
    请求-响应模式。适合仪表盘、用户侧查询与临时分析。
  </Card>
  <Card title="WebSocket" icon="tower-broadcast">
    亚秒级推送到浏览器、App 与需要实时 UI 更新的后端。
  </Card>
  <Card title="Kafka Streams" icon="bars-staggered">
    对交易引擎、索引器、告警流水线等延迟敏感后端的 exactly-once 投递。
  </Card>
</CardGroup>

### 内置合规

KYT（Know Your Transaction）与 KYA（Know Your Address）就在同一套 API 内。给一笔转账打分、给一个对手方画像、生成 Travel Rule 格式报告——不用第二个供应商、不用第二份账单、不多一个数据外泄面。

### 对 AI Agent 一等公民支持

AI Agent 与其他客户端使用相同接口接入 ChainStream。托管的 MCP Server 将我们的能力目录暴露为类型化工具；Agent Skills 以可安装的 SKILL.md 包形式发布；对 Agent 友好的鉴权（API Key、OAuth、x402/MPP 微支付）让 Agent 可以按次付费调用，无需预先开户。
