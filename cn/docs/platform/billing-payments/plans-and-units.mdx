## 概述

ChainStream 采用 **Unit 计费模型**。不同 API 消耗不同数量的 Unit，你可以选择适合业务需求的套餐。

## Unit 计算

### 什么是 Unit

Unit 是 ChainStream 的计费单位。不同 API 消耗不同数量的 Unit — 越复杂的 API 消耗越多。

### Data API 消耗

Unit 成本按计算复杂度分层。简单查询成本较低；聚合密集型和链上执行端点成本较高。

### WebSocket 用量估算示例

```
场景：监控 10 个代币的实时 K 线（1 分钟分辨率）

计算：
- 每条 K 线消息约 200 字节
- 每个代币每分钟约 1 条 K 线消息
- 10 代币 × 1 条/分 × 60 分 × 24 小时 × 30 天 = 432,000 条消息
- 总数据量：432,000 × 200 = 86,400,000 字节
- 月费：86,400,000 × 0.005 = 432,000 Units

场景：监控 50 个代币的统计数据（实时，约每秒 1 次更新）

计算：
- 代币统计消息约 500 字节
- 50 代币 × 1 条/秒 × 86,400 秒 × 30 天 = 129,600,000 条消息
- 总数据量：129,600,000 × 500 = 64,800,000,000 字节（约 64.8 GB）
- 月费：64,800,000,000 × 0.005 = 324,000,000 Units（约 324M）
```

<Note>
批量 API 通常有折扣。具体消耗请参考 API 参考文档。WebSocket 消息频率取决于市场活跃度；高波动时期消息量会增加。
</Note>

## 配额耗尽处理

### 配额用完会怎样

当月配额耗尽时，API 将返回 403 错误，无法继续调用。付费套餐用户可手动购买额外配额以恢复服务。

| 套餐 | 处理方式 |
|:--|:--|
| Free | API 返回 403 错误，等待下月配额重置 |
| Starter / Pro / Enterprise | API 返回 403 错误，可购买额外配额恢复服务 |
| Custom | 按合同条款 |

### 购买额外配额

## AI Agent 支付（x402 / MPP）

对于需要编程访问而无需 Dashboard 注册的 AI Agent，ChainStream 支持两种链上支付协议：

### x402（Base / Solana USDC）

[x402 协议](https://www.x402.org/) 实现 HTTP 原生微支付。当 API 调用返回 402 时，Agent 的钱包签名 USDC 转账并重试 — 无需注册账号。

| 属性 | 值 |
|------|-----|
| 网络 | Base（链 ID 8453）、Solana |
| 货币 | USDC |
| 标准 | EIP-3009（Base）、原生转账（Solana） |
| Gas 费 | **免费** — ChainStream 的 facilitator 代付所有 gas 费用 |

<Tip>
你只需支付 USDC 订阅费用。ChainStream 运营自有的 x402 facilitator，负责提交链上交易并代付 gas 费 — Agent 钱包无需持有 ETH 或 SOL 来支付 gas。
</Tip>

### MPP（Tempo USDC.e）

MPP 使用 Tempo 网络进行支付。Gas 直接用 USDC.e 支付 — 无需 ETH。

| 属性 | 值 |
|------|-----|
| 网络 | Tempo（链 ID 4217） |
| 货币 | USDC.e |

### 工作原理

1. Agent 调用任意 ChainStream API
2. 服务器返回 402（无有效订阅）
3. Agent 从 `/x402/pricing` 获取可用套餐
4. Agent 通过 x402 或 MPP 购买套餐
5. 服务器返回 **API Key** — Agent 保存用于后续所有调用

## 常见问题

<AccordionGroup>
  <Accordion title="如何选择合适的套餐？">
    建议先使用 Free 套餐评估实际用量，然后根据月消耗选择：
    - &lt; 30K Units：Free
    - 30K - 10M Units：Starter
    - 10M - 50M Units：Pro
    - &gt; 50M Units：Enterprise
  </Accordion>
  <Accordion title="可以随时升级/降级吗？">
    可以。升级立即生效，按比例收费。降级在下个计费周期生效。
  </Accordion>
  <Accordion title="未使用的配额会累积吗？">
    不会。月度配额在每月底重置，不会累积到下月。
  </Accordion>
  <Accordion title="如何获取发票？">
    在 Dashboard → 计费 → 发票中下载历史发票。Enterprise 客户可联系销售获取定制发票。
  </Accordion>
