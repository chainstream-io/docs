Pump.fun 是 Solana 上领先的 memecoin 发射台。ChainStream 通过 REST API、GraphQL、WebSocket 与 MCP 工具，为 Pump.fun 代币提供完整数据覆盖。

<Info>
**支持的能力**：代币发现、价格跟踪、成交分析、持仓者分析、通过 `pumpfun_create` 创建代币，以及毕业/迁移跟踪。
</Info>

---

## 如何发现新上线的 Pump.fun 代币？

使用排行榜接口查找 Pump.fun 发射台上新创建的代币。

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/ranking/sol/newTokens" \
  -H "X-API-KEY: your_api_key"
```

完整参数说明见 [Ranking - New Tokens](/cn/api-reference/endpoint/data/ranking/v2/ranking-chain-newtokens-get)。
  </Tab>
  <Tab title="GraphQL">

在 Solana 上查询 `TokenSupplyUpdates`，并筛选 Pump.fun 代币：

```graphql
query {
  TokenSupplyUpdates(
    network: sol
    limit: {count: 20}
    orderBy: Block_Time_DESC
  ) {
    Block { Time }
    TokenSupplyUpdate {
      Currency {
        MintAddress
        Decimals
      }
      PostBalance
      MarketCapInUSD
      PriceInUSD
      TotalSupply
    }
  }
}
```

<Tip>
在 [GraphQL IDE](https://ide.chainstream.io) 中打开即可交互式运行上述查询。
</Tip>
  </Tab>
  <Tab title="MCP Tool">
```
ranking_get_new_tokens({ chain: "solana" })
```

完整工具说明见 [MCP 工具目录](/cn/docs/ai-agents/mcp-server/tools)。
  </Tab>
</Tabs>

## 如何获取 Pump.fun 代币价格？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

完整说明见 [Token - Price](/cn/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

<Note>
将 `TOKEN_ADDRESS` 替换为实际的 Pump.fun 代币 mint 地址。
</Note>

## 如何获取 Pump.fun 代币的成交记录？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?tokenAddress=TOKEN_ADDRESS&limit=20" \
  -H "X-API-KEY: your_api_key"
```

完整说明见 [Trade - List](/cn/api-reference/endpoint/data/trade/v2/trade-chain-get)。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
        Account { Owner }
      }
      Dex { ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", tokenAddress: "TOKEN_ADDRESS", limit: 20 })
```
  </Tab>
</Tabs>

## 如何在 Pump.fun 上创建代币？

ChainStream 支持通过 API 直接在 Pump.fun 上创建代币。

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "pumpfun",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

完整说明见 [DEX - Create](/cn/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)。
  </Tab>
  <Tab title="MCP Tool">
```
pumpfun_create({
  name: "My Token",
  symbol: "MYTKN",
  description: "A test token"
})
```
  </Tab>
</Tabs>

<Note>
创建代币需要已充值的 Solana 钱包。私钥用于签名创建交易。
</Note>

## 常见问题

<AccordionGroup>
  <Accordion title="什么是 Pump.fun 毕业？">
    当 Pump.fun 代币的 bonding curve 达到 100% 时，代币会「毕业」——流动性迁移到 Raydium 池，代币开始在公开市场交易。使用 `ranking/sol/finalStretch` 接口跟踪接近毕业的代币，使用 `ranking/sol/migrated` 查看已毕业的代币。
  </Accordion>
  <Accordion title="如何过滤机器人/MEV 成交？">
    GraphQL 的 `DEXTrades` Cube 默认带有 `IsSuspect = false` 过滤，可自动排除疑似机器人与 MEV 成交。使用 REST API 时，请在成交响应中查看 `isSuspect` 字段。
  </Accordion>
  <Accordion title="能否实时跟踪 Pump.fun 代币？">
    可以。使用 WebSocket 订阅获取实时价格与成交流。配置方式见 [实时流式传输](/cn/docs/access-methods/websocket)。
  </Accordion>
  <Accordion title="哪些链支持 Pump.fun？">
    Pump.fun 仅支持 Solana。所有 Pump.fun 相关查询请使用 `chain: sol`（REST）或 `network: sol`（GraphQL）。
  </Accordion>
</AccordionGroup>
