ChainStream 目前支持 **3 条区块链** 的数据和 DeFi 操作。

## 链概览

| 链 | 标识符 | 类型 | 原生代币 | Data API | DeFi（兑换） | WebSocket | KYT/KYA |
|----|--------|------|---------|----------|-------------|-----------|---------|
| Solana | `sol` | L1 | SOL | 支持 | 支持 | 支持 | 支持 |
| BSC | `bsc` | L1 | BNB | 支持 | 支持 | 支持 | 支持 |
| Ethereum | `eth` | L1 | ETH | 支持 | 支持 | 支持 | 支持 |

<Note>
链标识符统一使用**小写**：`sol`、`eth`、`bsc`。
</Note>

## 各链 DEX 覆盖

| 链 | 支持的 DEX |
|----|-----------|
| Solana | Jupiter、Raydium、PumpFun、Moonshot、Candy |
| BSC | KyberSwap |
| Ethereum | KyberSwap |

<Note>
代币创建仅在 Solana 发射台上可用：PumpFun、Raydium。
</Note>

## 数据可用性

### 实时数据

三条链均提供：

- 代币元数据、统计和市场数据
- 钱包余额、盈亏和净值
- 交易历史和活动流
- 代币排行（热门、新币、已迁移、即将毕业）
- 持有者分析和分布
- 安全审计标志

### WebSocket 实时推送

所有链均支持实时订阅：

- 代币价格和统计更新
- OHLCV K 线数据流
- 交易活动流
- 钱包余额变动
- 新代币创建事件
- Bonding Curve 进度（Solana）

### DeFi 操作

| 操作 | Solana | Ethereum | BSC |
|------|--------|----------|-----|
| 兑换（报价 + 执行） | 支持 | 支持 | 支持 |
| 代币创建（发射台） | 支持 | -- | -- |
| 交易广播 | 支持 | 支持 | 支持 |
| Gas 估算 | -- | 支持 | 支持 |

## 各产品中的链标识符

| 场景 | Solana | BSC | Ethereum |
|------|--------|-----|----------|
| REST API | `sol` | `bsc` | `eth` |
| CLI `--chain` | `sol` | `bsc` | `eth` |
| SDK | `sol` | `bsc` | `eth` |
| MCP 工具 | `sol` | `bsc` | `eth` |
| WebSocket | `sol` | `bsc` | `eth` |
