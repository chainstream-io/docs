## 包

ChainStream MCP Server 以 npm 包形式分发：

```
@chainstream-io/mcp
```

## 安装

<Tabs>
  <Tab title="全局安装">
    全局安装以使 `chainstream-mcp` 二进制在系统范围内可用：

    ```bash
    npm install -g @chainstream-io/mcp
    ```

    安装后直接运行：

    ```bash
    chainstream-mcp
    ```
  </Tab>
  <Tab title="npx（免安装）">
    无需永久安装，按需运行 MCP Server：

    ```bash
    npx @chainstream-io/mcp
    ```

    每次下载并执行最新版本。适合快速测试。
  </Tab>
</Tabs>

## 二进制

该包暴露一个二进制：

```
chainstream-mcp
```

该二进制默认使用 stdio 传输启动 MCP Server — 这是连接 Claude Desktop、Cursor 和其他 MCP 客户端的标准模式。详细配置请参阅 [Stdio 传输](/cn/guides/ai-infrastructure/mcp-server/stdio-transport)。

## 环境变量

| 变量 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `CHAINSTREAM_API_KEY` | 是 | — | 你的 ChainStream API Key |
| `CHAINSTREAM_API_URL` | 否 | `https://api.chainstream.io` | ChainStream API 基础 URL |

将 API Key 设为环境变量或通过命令行传入：

```bash
export CHAINSTREAM_API_KEY=your-key
chainstream-mcp

# 或直接传入
chainstream-mcp --api-key your-key
```

<Warning>
  请妥善保管你的 API Key。切勿将其提交到版本控制。
</Warning>

## 验证安装

安装后验证包是否可用：

```bash
npx @chainstream-io/mcp --help
```

你应该看到可用选项：

```
Options:
  --transport  Transport type: stdio for local CLI   [choices: "stdio"] [default: "stdio"]
  --api-key    ChainStream API key                   [string]
  --help       Show help                             [boolean]
  --version    Show version number                   [boolean]
```
