## 多链数据 API

Chainstream 的多链数据 API 通过将来自不同区块链生态系统的资产持有和交易历史无缝聚合到统一的、易于使用的端点中，彻底改变了跨链资产组合管理。我们的 API 非常适合钱包服务、资产组合应用和分析仪表板，简化了多链资产追踪并提升了用户体验。
