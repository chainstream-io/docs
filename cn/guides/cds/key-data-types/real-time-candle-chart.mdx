## 实时 K 线图

Chainstream 的实时 K 线图为钱包、去中心化交易所（DEX）和分析平台直接提供准确的、流式的 OHLCV（开盘价、最高价、最低价、收盘价、交易量）数据。通过 API 和 WebSocket，Chainstream 确保用户始终能够即时访问每个主要区块链代币的全面价格走势。
