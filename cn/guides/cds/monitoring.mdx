## 监控

<Card>
  Chainstream 提供直观的个人资料页面，使您能够密切监控、分析和管理您的 API 使用情况。本指南将带您了解个人资料页面的主要功能，帮助您有效优化 API 集成。
</Card>

### 访问您的个人资料

<Steps>
  <Step title="登录">
    登录您的 Chainstream 账户
  </Step>
  <Step title="访问个人资料">
    点击右上角的用户名或头像
  </Step>
  <Step title="打开仪表板">
    从下拉菜单中选择个人资料以访问您的个人资料仪表板
  </Step>
</Steps>

### 监控 API 使用指标

<AccordionGroup>
  <Accordion title="API 使用指标" icon="chart-line" defaultOpen={true}>
    在您的个人资料中，您可以找到显示 API 总体使用情况的详细可视化指标，包括请求率、方法类型和状态码。这些洞察使您能够跟踪使用模式并主动管理 API 调用，以保持在订阅限制范围内。
  </Accordion>
</AccordionGroup>

### 查看 API 使用详情

<AccordionGroup>
  <Accordion title="使用详情" icon="list" defaultOpen={true}>
    探索详细日志，包括与您的 API 密钥关联的来源、IP 地址和用户代理。清楚了解 API 流量来源，确保您的 API 密钥得到安全和有效的使用。
  </Accordion>
</AccordionGroup>

### 查看审计日志

<AccordionGroup>
  <Accordion title="审计日志信息" icon="shield-check" defaultOpen={true}>
    审计日志提供全面的透明度，提供 API 密钥创建、修改、删除以及您账户内所有相关活动的时间顺序记录。定期查看审计日志有助于维护安全和负责任的 API 使用。
  </Accordion>
</AccordionGroup>
