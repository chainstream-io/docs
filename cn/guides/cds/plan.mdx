#### 总费用计算
要计算您的月度费用，请使用以下公式：

```text
总费用 = 基础价格 + 额外额度 × 超额费用
```

或更精确地：

```text
费用(X) = A(X) + [(T(X) - B(X)) / 1,000,000] × F(X)
```

其中：
- X = 套餐类型
- A(X) = X 套餐的基础价格
- B(X) = X 套餐包含的计算单位
- T(X) = 当月使用的总计算单位
- F(X) = X 套餐每增加 100万计算单位的费用
