## 安全检查

Chainstream 的安全检查通过分析多个安全指标来帮助您评估区块链代币的安全性和风险状况。每个安全标准都专门设计用于突出潜在的漏洞和风险，使用户能够根据清晰、可操作的洞察做出明智的决策。

### 解读结果

<Card>
Chainstream 清晰地标示每个标准的结果如下：

- **通过**：代币通过安全标准，未检测到重大风险。
- **警告**：检测到潜在风险；建议进一步调查。
- **失败**：代币未通过标准并构成重大安全风险。建议立即谨慎。
</Card>
