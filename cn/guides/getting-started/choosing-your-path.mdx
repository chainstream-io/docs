## 数据用户

### 适用场景

需要获取链上数据用于展示、分析或交易决策的开发者。

**典型用例：**
- 在 dApp 中展示代币实时价格
- 构建行情看板或数据分析工具
- 监控特定钱包的持仓变化
- 开发交易机器人获取市场数据

**典型用户：**
- DeFi 应用开发者
- 行情数据展示平台
- 链上数据分析师
- 交易机器人开发者

### 推荐路径

<Steps>
  <Step title="理解数据模型">
    阅读 [数据概念](/cn/guides/data-concepts/realtime-streaming) 了解 ChainStream 的数据结构
  </Step>
  <Step title="调用代币 API">
    查看 [Token API 参考](/cn/api-reference/endpoint/token/v1/token-chain-tokenaddress-get) 获取价格和代币信息
  </Step>
  <Step title="实战：价格预警">
    跟随 [价格预警机器人教程](/cn/playbooks/tutorials/build-price-alert-bot) 构建完整应用
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- 实时价格数据（WebSocket 推送）
- 多链代币信息查询
- 交易和持仓数据
- Smart Money 钱包追踪

<Note>
**预计时间**：完成推荐路径约需 1-2 小时
</Note>

## GraphQL 分析师

### 适用场景

需要对链上 OLAP 数据进行灵活、声明式查询并具备聚合能力的开发者和分析师。

**典型用例：**
- 分析多链 DEX 交易模式
- 用聚合的链上指标构建自定义看板
- 查询代币持有人分布和钱包 PnL
- 对交易、转账和池进行即席分析查询

**典型用户：**
- 链上数据分析师
- 量化研究员
- 看板和分析工具开发者
- DeFi 协议团队

### 推荐路径

<Steps>
  <Step title="了解 GraphQL API">
    阅读 [GraphQL 概览](/cn/graphql/getting-started/overview) 了解 OLAP 分析型 API
  </Step>
  <Step title="运行第一个查询">
    跟随 [第一个查询指南](/cn/graphql/getting-started/first-query) 使用 GraphQL IDE
  </Step>
  <Step title="探索 Data Cubes">
    了解 [12 个 Data Cubes](/cn/graphql/schema/cubes)，涵盖交易、转账、余额等
  </Step>
  <Step title="尝试查询示例">
    浏览 [DEX 交易示例](/cn/graphql/examples/dex-trades) 和其他查询示例
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- 单一 GraphQL 端点查询所有链上数据
- 12 个 Data Cubes 涵盖 DEX 交易、转账、余额、池、OHLC 和代币指标
- 内置聚合功能（count、sum、avg、min、max、uniq）
- 灵活的过滤、排序和分页
- 浏览器端 GraphQL IDE，内置模板和代码导出

<Note>
**预计时间**：完成推荐路径约需 1-2 小时
</Note>

## 合规用户

### 适用场景

需要评估交易风险或分析地址安全性的团队。

**典型用例：**
- 检测用户充值是否来自高风险地址
- 分析钱包地址的历史行为
- 生成交易风险评估报告
- 集成风控流程到业务系统

**典型用户：**
- 交易所合规团队
- 钱包安全团队
- 风控系统开发者

### 推荐路径

<Steps>
  <Step title="理解 KYT/KYA">
    阅读 [安全合规概述](/cn/guides/security-compliance/overview) 了解风控能力
  </Step>
  <Step title="调用风险评估 API">
    查看 [KYT API 参考](/cn/api-reference/endpoint/kyt/v1/kyt-transfer-post) 了解交易风险评估接口
  </Step>
  <Step title="实战：充值风控">
    跟随 [充值风控集成教程](/cn/playbooks/tutorials/integrate-deposit-check) 实现完整流程
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- KYT 交易风险评估（返回风险分数和标签）
- KYA 地址画像分析（地址类型、历史行为）
- 风险标签体系（可配置的风险规则）

<Note>
**预计时间**：完成推荐路径约需 2-3 小时
</Note>

## AI 开发者

### 适用场景

希望让 AI Agent 能够查询和理解链上数据的开发者。

**典型用例：**
- 让 Claude/GPT 回答链上数据问题
- 构建能理解加密市场的 AI 助手
- 在 AI 工作流中集成链上数据查询

**典型用户：**
- AI Agent 开发者
- 自动化交易系统开发者
- LLM 应用开发者
- MCP 集成开发者

### 推荐路径

<Steps>
  <Step title="理解 AI 能力">
    阅读 [AI 基础设施概述](/cn/guides/ai-infrastructure/overview) 了解 MCP 协议
  </Step>
  <Step title="配置 MCP Server">
    按照 [MCP 配置指南](/cn/guides/ai-infrastructure/mcp-server/setup-guide) 完成接入
  </Step>
  <Step title="实战：AI 助理">
    跟随 [AI 交易助理教程](/cn/playbooks/tutorials/ai-agent-with-mcp) 构建完整应用
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- MCP Server 集成（支持 Claude Desktop 等客户端）
- AI 友好的工具集（结构化输出）
- 自然语言查询（如"查询 ETH 最近 24 小时价格"）

<Note>
**预计时间**：完成推荐路径约需 1-2 小时
</Note>

## AI Agent 开发者

### 适用场景

构建需要数据访问和 DeFi 执行能力的 AI Agent，通过 CLI 或 Agent Skills 实现。

**典型用例：**
- 构建能查询链上数据并执行交易的 AI Agent
- 自动化代币研究和投资组合管理
- 将 ChainStream 集成到 CI/CD 流水线
- 在 Cursor、Claude Code 或 Codex 中使用 Agent Skills

**典型用户：**
- AI Agent 开发者
- 自动化链上工作流的 DevOps 工程师
- Cursor / Claude Code 高级用户

### 推荐路径

<Steps>
  <Step title="安装 CLI">
    阅读 [CLI 概览](/cn/guides/cli/overview) 并运行 `npx @chainstream-io/cli login`
  </Step>
  <Step title="安装 Agent Skills">
    按照 [Skills 安装指南](/cn/guides/ai-infrastructure/agent-skills/installation) 配置你的平台
  </Step>
  <Step title="探索数据查询">
    学习 [chainstream-data skill](/cn/guides/ai-infrastructure/agent-skills/chainstream-data) 进行代币研究和钱包分析
  </Step>
  <Step title="执行 DeFi（可选）">
    学习 [chainstream-defi skill](/cn/guides/ai-infrastructure/agent-skills/chainstream-defi) 进行兑换和代币创建
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- CLI 命令用于代币、市场、钱包和 DeFi 操作
- Agent Skills 用于结构化 AI 工作流
- x402 自动支付实现无缝订阅管理
- MCP Server 集成用于 AI 聊天助手

<Note>
**预计时间**：完成推荐路径约需 30-60 分钟
</Note>
