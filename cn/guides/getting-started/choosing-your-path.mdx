## 数据用户

### 适用场景

需要获取链上数据用于展示、分析或交易决策的开发者。

**典型用例：**
- 在 dApp 中展示代币实时价格
- 构建行情看板或数据分析工具
- 监控特定钱包的持仓变化
- 开发交易机器人获取市场数据

**典型用户：**
- DeFi 应用开发者
- 行情数据展示平台
- 链上数据分析师
- 交易机器人开发者

### 推荐路径

<Steps>
  <Step title="理解数据模型">
    阅读 [数据概念](/cn/guides/data-concepts/realtime-streaming) 了解 ChainStream 的数据结构
  </Step>
  <Step title="调用代币 API">
    查看 [Token API 参考](/cn/api-reference/endpoint/token/v1/token-chain-tokenaddress-get) 获取价格和代币信息
  </Step>
  <Step title="实战：价格预警">
    跟随 [价格预警机器人教程](/cn/playbooks/tutorials/build-price-alert-bot) 构建完整应用
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- 实时价格数据（WebSocket 推送）
- 多链代币信息查询
- 交易和持仓数据
- Smart Money 钱包追踪

<Note>
**预计时间**：完成推荐路径约需 1-2 小时
</Note>

## 合规用户

### 适用场景

需要评估交易风险或分析地址安全性的团队。

**典型用例：**
- 检测用户充值是否来自高风险地址
- 分析钱包地址的历史行为
- 生成交易风险评估报告
- 集成风控流程到业务系统

**典型用户：**
- 交易所合规团队
- 钱包安全团队
- 风控系统开发者

### 推荐路径

<Steps>
  <Step title="理解 KYT/KYA">
    阅读 [安全合规概述](/cn/guides/security-compliance/overview) 了解风控能力
  </Step>
  <Step title="调用风险评估 API">
    查看 [KYT API 参考](/cn/api-reference/endpoint/kyt/v1/kyt-transfer-post) 了解交易风险评估接口
  </Step>
  <Step title="实战：充值风控">
    跟随 [充值风控集成教程](/cn/playbooks/tutorials/integrate-deposit-check) 实现完整流程
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- KYT 交易风险评估（返回风险分数和标签）
- KYA 地址画像分析（地址类型、历史行为）
- 风险标签体系（可配置的风险规则）

<Note>
**预计时间**：完成推荐路径约需 2-3 小时
</Note>

## AI 开发者

### 适用场景

希望让 AI Agent 能够查询和理解链上数据的开发者。

**典型用例：**
- 让 Claude/GPT 回答链上数据问题
- 构建能理解加密市场的 AI 助手
- 在 AI 工作流中集成链上数据查询

**典型用户：**
- AI Agent 开发者
- 自动化交易系统开发者
- LLM 应用开发者
- MCP 集成开发者

### 推荐路径

<Steps>
  <Step title="理解 AI 能力">
    阅读 [AI 基础设施概述](/cn/guides/ai-infrastructure/overview) 了解 MCP 协议
  </Step>
  <Step title="配置 MCP Server">
    按照 [MCP 配置指南](/cn/guides/ai-infrastructure/mcp-server/setup-guide) 完成接入
  </Step>
  <Step title="实战：AI 助理">
    跟随 [AI 交易助理教程](/cn/playbooks/tutorials/ai-agent-with-mcp) 构建完整应用
  </Step>
</Steps>

### 核心能力

你将使用到以下能力：
- MCP Server 集成（支持 Claude Desktop 等客户端）
- AI 友好的工具集（结构化输出）
- 自然语言查询（如"查询 ETH 最近 24 小时价格"）

<Warning>
**注意**：当前 AI 执行层仅支持读取操作，不支持自动执行交易等写入操作。
</Warning>

<Note>
**预计时间**：完成推荐路径约需 1-2 小时
</Note>
