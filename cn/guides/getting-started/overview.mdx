## 什么是 ChainStream

ChainStream 是为 Web3 时代构建的 AI-Native 基础设施。与传统 Web3 数据 API 不同，ChainStream 从第一天起就为 AI Agent 和自动化场景设计，提供：

- **统一数据层**: 一个 API 获取多链数据，无需对接多个供应商
- **内置合规能力**: KYT/KYA 原生集成，无需第三方风控服务
- **AI 执行层**: 通过 MCP 协议让 AI Agent 直接操作链上世界

<Tip>
让开发者和 AI Agent 能够安全、高效地访问和操作链上世界。
</Tip>

## 核心能力

ChainStream 提供三大核心能力，覆盖 Web3 数据获取、安全合规和 AI 执行的完整链路：

### Data — 实时链上数据

通过统一 API 访问多链实时数据，支持 WebSocket 实时推送和历史数据查询。

<AccordionGroup>
  <Accordion title="多链数据聚合">
    一个 API 端点获取 Ethereum、Solana、BSC、Polygon 等 10+ 主流公链数据，无需分别对接多个数据源。支持跨链查询和数据标准化。
  </Accordion>
  <Accordion title="实时价格与交易数据">
    毫秒级价格更新，支持 DEX/CEX 聚合价格、交易对深度、K线数据。提供 WebSocket 实时推送，延迟 &lt;2ms。
  </Accordion>
  <Accordion title="Smart Money 钱包追踪">
    追踪链上聪明钱动向，包括鲸鱼钱包、机构地址、KOL 钱包。提供钱包标签、持仓变化、交易行为分析。
  </Accordion>
  <Accordion title="历史数据查询">
    支持任意时间范围的历史数据查询，包括历史价格、历史持仓、历史交易记录。数据保留期限根据套餐不同。
  </Accordion>
</AccordionGroup>

### Security — 安全与合规

内置企业级安全合规能力，满足交易所、钱包、DeFi 协议的风控需求。

<AccordionGroup>
  <Accordion title="KYT (Know Your Transaction) 交易风险评估">
    实时分析交易风险等级，识别可疑资金流向。支持 OFAC 制裁名单检查、混币器识别、暗网关联检测。返回风险评分 (0-100) 和详细风险标签。
  </Accordion>
  <Accordion title="KYA (Know Your Address) 地址画像分析">
    全面的地址画像分析，包括地址类型识别（EOA/合约/多签）、历史行为分析、资金来源追溯、关联地址图谱。
  </Accordion>
  <Accordion title="合规审计支持">
    提供完整的审计日志，满足监管报告需求。支持导出 FATF Travel Rule 格式报告，集成主流合规工具（Chainalysis、Elliptic 格式兼容）。
  </Accordion>
</AccordionGroup>

### AI Execution — AI 执行层

通过 MCP (Model Context Protocol) 让 AI Agent 安全地与区块链交互。

<AccordionGroup>
  <Accordion title="MCP Server 支持">
    原生支持 Anthropic MCP 协议，让 Claude 等 AI 模型直接调用 ChainStream 能力。无需编写适配代码，开箱即用。
  </Accordion>
  <Accordion title="AI Agent 友好的工具集">
    专为 AI Agent 设计的工具接口，包括结构化输出、错误自愈、上下文保持。支持 LangChain、AutoGPT、CrewAI 等主流框架。
  </Accordion>
  <Accordion title="自然语言到链上操作">
    AI 可通过自然语言指令完成链上查询和操作，如"查询 0x... 地址最近 7 天的交易"、"监控 PEPE 价格超过 $0.00001 时提醒我"。
  </Accordion>
</AccordionGroup>

<Warning>
**安全提示**: AI 执行层仅支持读取操作和预授权的写入操作，所有敏感操作需要用户确认。
</Warning>
