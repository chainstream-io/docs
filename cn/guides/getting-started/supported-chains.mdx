## 网络类型

### EVM（以太坊虚拟机）

链：Ethereum、BSC、Polygon、Arbitrum、Base、Avalanche、Optimism

- 使用标准 `{chain}` 参数：`eth`、`bsc`、`polygon`、`arbitrum`、`base`、`avax`、`optimism`
- 兼容 ERC-20 代币标准
- 地址格式：`0x` 前缀，42 位十六进制字符串

### SVM（Solana 虚拟机）

链：Solana

- 使用 `sol` 作为 chain 参数
- SPL 代币标准
- 地址格式：Base58 编码，32-44 个字符

### TVM（TRON 虚拟机）

链：TRON

- 使用 `tron` 作为 chain 参数
- TRC-20 代币标准
- 地址格式：Base58Check 编码，以 `T` 开头

## Kafka Streams 各链支持

Kafka Streams 提供最低延迟的数据管线，目前支持：

| 链 | 可用 Topics | 延迟 |
|:--|:--|:--|
| Solana | dex.trades, tokens, balances, dex.pools, transfers, candlesticks | < 500ms |
| Ethereum | dex.trades, tokens, balances, dex.pools, transfers, candlesticks | < 2s |
| BSC | dex.trades, tokens, balances, dex.pools, transfers, candlesticks | < 2s |
| TRON | dex.trades, tokens, balances, transfers | < 2s |

所有链还支持衍生 Topics：`token-supplies`、`token-prices`、`token-holdings`、`token-market-caps`、`trade-stats`。

<Tip>
Kafka Streams 需要单独的凭证。联系 [support@chainstream.io](mailto:support@chainstream.io) 申请访问。
</Tip>

## API Base URL

所有链使用同一个 Base URL，通过 `{chain}` 路径参数指定链：

```
https://api.chainstream.io/v2/{resource}/{chain}/...
```

**示例：**

```bash
# Solana Token 元数据
GET https://api.chainstream.io/v2/token/sol/{tokenAddress}/metadata

# Ethereum Token 价格
GET https://api.chainstream.io/v2/token/eth/{tokenAddress}/price

# BSC 钱包余额
GET https://api.chainstream.io/v2/wallet/bsc/{walletAddress}/tokens/balance
```
