## 隐私政策

<Info>
Chainstream（"我们"、"我们的"或"我们公司"）提供实时区块链数据流平台，服务对象包括开发者、分析师和机构。本隐私政策旨在帮助您了解我们如何收集、使用和披露您的个人信息，以及您如何行使隐私权。
</Info>

### 适用范围

<Note>
本隐私政策适用于我们通过网站、文档门户、API、终端、SDK 和任何相关服务（统称为"服务"）收集的个人信息。
</Note>

### 政策变更

<Note>
我们可能随时修改本隐私政策。如有重大变更，我们将通过电子邮件通知您或更新政策日期。您继续使用我们的服务即表示您接受更新后的条款。
</Note>

### 信息收集

<CardGroup cols={1}>
  <Card title="您提供的信息" icon="user">
    - 账户注册：当您注册 Chainstream 账户时，我们会收集您的姓名、电子邮件、组织名称（可选）、密码和区块链钱包地址（如适用）。
    - API 使用：当您请求 API 密钥或访问付费功能时，我们可能会收集账单信息和支付详情。
    - 支持和通信：如果您联系我们寻求支持或提供反馈，我们会收集您的通信内容和您提供的任何其他数据。
    - 代币提交：如果您提交代币更新或使用我们的代币信息更新服务，我们可能会收集元数据、代币图片、社交链接和相关信息。
  </Card>

  <Card title="自动收集的信息" icon="robot">
    - 日志数据：包括 IP 地址、浏览器类型、操作系统、引用 URL、设备标识符、时间戳、API 调用指标和用户代理。
    - 使用分析：我们自动跟踪 API 使用情况（包括主题订阅、查询量和响应时间）以改进性能并确保公平使用。
    - Cookie 和跟踪技术：我们使用 cookie 和类似技术来增强用户体验、分析使用情况和个性化内容。这包括必要的、分析性的和偏好性的 cookie。
  </Card>

  <Card title="第三方信息" icon="link">
    - 集成：当您将 Chainstream 账户与第三方工具（如钱包、交易所）集成时，我们可能会根据这些服务的许可接收相关数据。
    - 公开可用的区块链数据：我们索引和关联已经公开可访问的区块链数据，包括钱包地址和交易历史。
  </Card>
</CardGroup>

### 信息使用

<Card title="数据使用目的" icon="chart-mixed">
我们使用收集的信息来：
- 提供、维护和改进我们的服务
- 管理用户账户和验证登录
- 监控和执行 API 速率限制和公平使用
- 响应用户查询并提供技术支持
- 发送服务相关通知和通信
- 处理账单和订阅相关任务
- 进行内部分析和开发新功能
- 执行我们的服务条款并防止欺诈
</Card>

### 您的隐私选择和权利

<CardGroup cols={1}>

    - 请求更正或删除您的数据
    - 反对或限制某些数据使用
    - 请求数据可携带性

    要行使这些权利，请联系 privacy@chainstream.io。我们将验证您的身份并按法律要求作出回应。
  </Card>
</CardGroup>
