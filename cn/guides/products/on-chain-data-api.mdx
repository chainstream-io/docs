<Info>
ChainStream 的链上数据 API 是您获取主要网络上可操作的代币、流动池、合约和地址数据的门户。无论您是构建 DEX、钱包、风险引擎还是分析平台，此 API 都为您提供支持智能决策和应用内洞察所需的基础层。
</Info>

## 为什么使用 ChainStream 链上数据？

区块链数据功能强大，但杂乱无章。开发人员经常需要解析原始日志或编写自定义索引器，仅仅是为了回答简单的问题——这个代币的价格是多少？这个地址是否可疑？这个流动池是否有流动性？

ChainStream 通过提供干净、随时可查询的接口来访问定义链上行为的核心对象，解决了这一问题。您无需解码合约或追踪特定链的格式，即可获得针对性能和清晰度优化的结构化数据。
