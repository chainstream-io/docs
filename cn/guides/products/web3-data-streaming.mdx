<Info>
Chainstream 的 Web3 数据流服务提供对以太坊、Solana、BNB Chain 和 Polygon 等主要链上活动的快速、可靠访问。
</Info>

## 为什么使用 Web3 数据流？

传统 API 需要不断轮询且经常错过短暂事件。通过 Chainstream，您可以在区块链活动发生时获得基于推送的实时访问。无论您是构建 DeFi 监控工具、自动交易机器人还是预警系统，流数据都为高性能、事件驱动的基础设施提供了基础。
