**零延迟。零噪音。仅在关键时刻和地点提供可操作的数据。**

## 核心构建 API

<AccordionGroup>
  <Accordion title="Web3 数据流服务" icon="satellite-dish" defaultOpen>
    获取结构化的链上事件—通过 Kafka、WebSocket 或您首选的方式传递。包括代币转账、交换、NFT 活动、合约调用和自定义过滤器。
  </Accordion>
  
  <Accordion title="链上数据 API" icon="database">
    用代币元数据、地址分类、合约情报和资金池统计丰富警报内容。
  </Accordion>
  
  <Accordion title="账户和资产 API" icon="wallet">
    当警报触发时，立即获取该钱包所拥有的资产—代币、NFT、仓位—无需编写多链查询。
  </Accordion>
  
  <Accordion title="GraphQL 查询引擎" icon="brackets-curly">
    创建自定义分析查询：例如本小时最大流入地址、按规模划分的鲸鱼交换路径，或按协议划分的最近活动峰值。
  </Accordion>
</AccordionGroup>
