<Info>
构建 DeFi 产品不仅仅是部署智能合约。开发者需要快速可靠的数据来跟踪协议事件、监控资产流动并触发时间敏感的操作。无论您是构建借贷协议、质押应用还是衍生品平台，访问实时链上数据对用户安全、合约逻辑和用户体验质量都至关重要。
</Info>

## ChainStream 如何提供帮助

ChainStream 提供 DeFi 协议所需的低延迟数据层，使其即使在网络压力或市场波动期间也能自信运行。

<CardGroup cols={2}>
  <Card title="实时事件推送" icon="satellite-dish" color="#4D9CFF">
    不再需要轮询状态变化。使用事件流实时响应存款、兑换、领取和预言机更新。
  </Card>
  
  <Card title="账户级资产快照" icon="wallet" color="#16A34A">
    通过单个 API 调用，准确了解钱包在各个区块链、代币、资金池和协议中的持有情况。
  </Card>
  
  <Card title="Kafka 驱动的性能" icon="gauge-high" color="#FB6514">
    即使在波动期间，也能以一致的吞吐量和低延迟处理大量事件。
  </Card>
  
  <Card title="GraphQL 智能查询" icon="brackets-curly" color="#9333EA">
    无需链式请求，即可提出关于资金池状态、地址历史或代币组成的复杂问题。
  </Card>
</CardGroup>

## DeFi 核心 API

<AccordionGroup>
  <Accordion title="Web3 数据流服务" icon="satellite-dish" defaultOpen>
    通过 Kafka、WebSocket 或您首选的管道订阅高频区块链事件（存款、清算、兑换等）。
  </Accordion>
  
  <Accordion title="账户和资产 API" icon="wallet">
    查询跨 25+ 条链的实时资产持有情况，包括代币余额、DeFi 仓位和质押奖励。
  </Accordion>
  
  <Accordion title="链上数据 API" icon="database">
    检索标准化的资金池数据、智能合约事件日志和地址级情报，以支持协议操作。
  </Accordion>
  
  <Accordion title="GraphQL 查询引擎" icon="brackets-curly">
    为金库统计、用户风险敞口和流动性跟踪构建精确、高效的跨链查询。
  </Accordion>
</AccordionGroup>
