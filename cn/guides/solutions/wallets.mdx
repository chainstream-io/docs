## 可用的构建 API

<AccordionGroup>
  <Accordion title="账户和资产 API" icon="wallet" defaultOpen>
    钱包视图的核心。检索跨链的完整持有情况：代币余额、NFT 所有权、DeFi 仓位、协议元数据。
  </Accordion>
  
  <Accordion title="Web3 数据流服务" icon="satellite-dish">
    订阅代币转账、NFT 铸造/销售、协议交互—按钱包地址、合约或事件类型过滤。
  </Accordion>
  
  <Accordion title="链上数据 API" icon="database">
    使用代币信息、NFT 元数据和协议标签丰富持有信息。为余额添加上下文：价格、符号、风险标签。
  </Accordion>
  
  <Accordion title="GraphQL 查询引擎" icon="brackets-curly">
    查询自定义钱包行为—如"过去一周的热门 NFT 转账"或"该地址在过去 24 小时内收到的新代币"。
  </Accordion>
</AccordionGroup>
