## Installation

```bash
go get github.com/chainstream-io/chainstream-go-sdk
```

## Initialization

```go
package main

import (
    "log"
    "github.com/chainstream-io/chainstream-go-sdk"
)

func main() {
    // Initialize the client
    client, err := chainstream.NewChainStreamClient("YOUR_ACCESS_TOKEN", &chainstream.ChainStreamOptions{
        ServerUrl: "https://api-dex.chainstream.io",
        StreamUrl: "wss://realtime-dex.chainstream.io/connection/websocket",
    })
    if err != nil {
        log.Fatal(err)
    }
    defer client.Close()
}
```

## Basic Usage

### Query Token Information
```go
// Get token details
tokenInfo, err := client.Token.GetToken(context.Background(), "sol", "6p6xgHyF7AeE6TZkSmFsko444wqoP15icUSqi2jfGiPN")
if err != nil {
    log.Fatal(err)
}
```

### Subscribe to Real-time Data
```go
// Just subscribe directly, SDK auto-detects connection status and connects if needed
unsubscribe := client.Stream.SubscribeTokenCandles(
    "sol",
    "6p6xgHyF7AeE6TZkSmFsko444wqoP15icUSqi2jfGiPN",
    "1m",
    func(candle api.TokenCandle) {
        log.Printf("New candle: %+v", candle)
    },
    "", // filter (optional)
)
defer unsubscribe()
```

<Note>
When calling `SubscribeXXX` methods, SDK automatically checks WebSocket connection status and establishes connection if not connected. No need to manually call `Connect()`.
</Note>

## Configuration Options

<ParamField path="accessToken" type="string" required>
  API access token
</ParamField>

<ParamField path="options.ServerUrl" type="string">
  API server URL (default: https://api-dex.chainstream.io)
</ParamField>

<ParamField path="options.StreamUrl" type="string">
  WebSocket stream URL (default: wss://realtime-dex.chainstream.io/connection/websocket)
</ParamField>

<ParamField path="options.Debug" type="bool">
  Enable debug mode
</ParamField>

<ParamField path="options.AutoConnectWebSocket" type="bool">
  Whether to auto-connect WebSocket on initialization (default: false). Usually not needed as subscription methods handle connection automatically.
</ParamField>

## Token Provider

For dynamic token management, you can use a token provider:

```go
type MyTokenProvider struct {
    // Your token provider implementation
}

func (p *MyTokenProvider) GetToken() (string, error) {
    // Return current token
    return "your-current-token", nil
}

// Initialize with token provider
client, err := chainstream.NewChainStreamClientWithTokenProvider(&MyTokenProvider{}, &chainstream.ChainStreamOptions{})
```
