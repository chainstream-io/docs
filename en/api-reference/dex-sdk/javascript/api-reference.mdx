This document describes the methods provided by the ChainStream DEX JavaScript SDK. Each method is a wrapper for the corresponding REST API. For detailed parameter descriptions and return value structures, please click the corresponding API link.

## DEX Operations

### List DEXes
Get a list of all DEXes on the specified chain.

<Tip>
  **REST API:** [DEX - List](/en/api-reference/endpoint/dex/dex)
</Tip>

```typescript
client.dex.list()
```

### Token Minting
Create a new token on the specified DEX.

<Tip>
  **REST API:** [DEX - Mint](/en/api-reference/endpoint/dex/dex-chain-mint)
</Tip>

```typescript
client.dex.mint()
```

### Swap
Swap tokens.

<Tip>
  **REST API:** [DEX - Swap](/en/api-reference/endpoint/dex/dex-chain-swap)
</Tip>

```typescript
client.dex.swap()
```

### Get Swap Route
Calculate the best route for token swaps.

<Tip>
  **REST API:** [DEX - Route](/en/api-reference/endpoint/dex/dex-chain-route)
</Tip>

```typescript
client.dex.route()
```
