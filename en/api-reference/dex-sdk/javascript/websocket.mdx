This document describes the WebSocket subscription methods provided by the ChainStream DEX JavaScript SDK. Each method is a wrapper for the corresponding WebSocket API. For detailed parameter descriptions and return value structures, please click the corresponding API link.

## Candlestick Data Subscription

### Subscribe to Candlestick Data
Subscribe to token price candlestick data.

<Tip>
  **WebSocket API:** [Candlestick Data](/en/api-reference/endpoint/websocket/api#candlestick-data)
</Tip>

```typescript
client.stream.subscribeTokenCandles()
```

## Wallet Data Subscription

### Subscribe to Wallet Balance
Subscribe to wallet balance information.

<Tip>
  **WebSocket API:** [Wallet Balance](/en/api-reference/endpoint/websocket/api#wallet-balance)
</Tip>

```typescript
client.stream.subscribeWalletBalance()
```

### Subscribe to Token Balance
Subscribe to balance information for a specific token.

<Tip>
  **WebSocket API:** [Token Balance](/en/api-reference/endpoint/websocket/api#token-balance)
</Tip>

```typescript

```

## Connection Management

### Establish Connection
```typescript
client.stream.connect()
```

All subscription methods return an `Unsubscribe` object that can be used to cancel the subscription:

```typescript
const subscription = client.stream.subscribeTokenCandles(/* ... */);
// Cancel subscription
subscription.unsubscribe();
```

For detailed parameter descriptions, data formats, and error handling, please refer to the [WebSocket API Documentation](/en/api-reference/endpoint/websocket/api).
