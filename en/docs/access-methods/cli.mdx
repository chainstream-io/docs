The ChainStream CLI is the command-line front door to the same [data products](/en/docs/data-products/overview) you'd hit via SDK, REST or GraphQL. It's designed to compose with `jq`, shell pipelines and AI agents that drive shells.

## Install

```bash
# macOS / Linux — via Homebrew
brew install chainstream-io/tap/chainstream

# npm
npm install -g @chainstream-io/cli

# Direct binary
curl -fsSL https://chainstream.io/install.sh | sh
```

## Authenticate

```bash
chainstream auth login          # opens browser for OAuth
# or
export CHAINSTREAM_API_KEY=...  # use env var for scripts
```

## First commands

```bash
chainstream tokens search --chain solana --query USDC --limit 5
chainstream trades tail --chain ethereum --token 0xA0b8...
chainstream rankings hot --chain solana --window 24h
chainstream defi quote --chain solana --from SOL --to USDC --amount 1
```

Output is JSON by default; pass `--format=table` for humans.

## When to use

- Quick exploration ("what's happening right now?")
- Scripts, cron jobs, GitHub Actions
- Giving an AI agent shell-executable ChainStream primitives
- Combining ChainStream output with other CLI tools via pipes
