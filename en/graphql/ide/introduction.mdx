## IDE Layout

The IDE is organized into five regions:

```
┌──────────────────────────────────────────────────┐
│  Toolbar  [▶ Execute] [Prettify] [Copy] [Merge]  │
│           [Save Query] [Export Code]              │
├────────────┬─────────────────────────────────────┤
│            │                                     │
│  Sidebar   │         Query Editor                │
│            │                                     │
│ ┌────────┐ │─────────────────────────────────────│
│ │Queries │ │  Variables  │  Headers              │
│ │Saved   │ │─────────────────────────────────────│
│ └────────┘ │                                     │
│            │         Results Panel               │
│            │                                     │
├────────────┴─────────────────────────────────────┤
│  Status: Ready  │  120ms  │  2.4 KB  │  8 CU     │
└──────────────────────────────────────────────────┘
```

| Region | Description |
|:--|:--|
| **Toolbar** | Execute, Prettify, Copy, Merge, Save Query, and Export Code buttons |
| **Sidebar** | Two tabs — **Queries** (built-in templates by chain → Cube) and **Saved** (your locally saved queries) |
| **Query Editor** | Monaco-powered GraphQL editor with autocomplete and error markers |
| **Variables / Headers** | Bottom panels for JSON variables and HTTP headers (Headers shown by default with `X-API-KEY`) |
| **Results Panel** | Syntax-highlighted JSON response from the GraphQL endpoint |
| **Status Bar** | Request state, latency, response size, and Credit Units (CU) consumed |

## Quick Start

<Steps>
  <Step title="Open the IDE">
    Navigate to [ide.chainstream.io](https://ide.chainstream.io) in your browser.
  </Step>
  <Step title="Set your API Key">
    Click the **Headers** panel at the bottom of the editor and enter your API Key:
    ```json
    {
      "X-API-KEY": "your_api_key"
    }
    ```
    <Tip>
    Don't have a key yet? Create one in the [ChainStream Dashboard](https://www.chainstream.io/dashboard) → **Applications** → **Create New App**.
    </Tip>
  </Step>
  <Step title="Load a template">
    Open the sidebar (**Ctrl/Cmd+B**), expand a chain (e.g. **Solana**), then click a template like **DEXTrades → Latest Trades**. The query loads into the editor with placeholder values.
  </Step>
  <Step title="Replace placeholders">
    Replace any `TOKEN_ADDRESS`, `WALLET_ADDRESS`, or `POOL_ADDRESS` placeholders with real on-chain addresses.
  </Step>
  <Step title="Execute">
    Press **Ctrl/Cmd+Enter** or click the **Execute** (▶) button. Results appear in the right panel, and the status bar shows latency and CU cost.
  </Step>
</Steps>
