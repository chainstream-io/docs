## Overview

ChainStream provides infrastructure specifically designed for AI applications, enabling AI Agents to directly query on-chain data, analyze market information, and even execute trading operations.

## Why AI Needs On-chain Data

Traditional AI models (like ChatGPT, Claude) have knowledge cutoffs and cannot access real-time on-chain information. ChainStream AI infrastructure solves this:

<AccordionGroup>
  <Accordion title="Real-time Data Access" icon="bolt">
    AI can access latest:
    - Token prices and trading volume
    - Wallet balances and holdings
    - DEX liquidity and trades
    - Market rankings
  </Accordion>
  
  <Accordion title="On-chain Analysis" icon="chart-mixed">
    AI can perform:
    - Token security checks
    - Wallet PnL calculations
    - Smart Money tracking
    - Risk assessments
  </Accordion>
  
  <Accordion title="Trade Assistance" icon="right-left">
    AI can help users:
    - Query optimal trade routes
    - Estimate gas fees
    - Build transactions (requires user confirmation)
  </Accordion>
</AccordionGroup>

## MCP (Model Context Protocol)

MCP is an open protocol allowing AI models to interact with external data sources and tools. ChainStream provides an MCP Server implementation supporting major AI models.

### Supported Models

| Model | Support Status | Notes |
|-------|----------------|-------|
| Claude | Native Support | Official Anthropic MCP support |
| GPT-4 | Via Function Calling | OpenAI function calling |
| Cursor AI | Native Support | IDE built-in MCP |
| Others | In Progress | Contact us for details |
