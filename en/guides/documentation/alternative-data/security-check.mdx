## Security Check

Chainstream's Security Check helps you evaluate the safety and risk profile of blockchain tokens by analyzing multiple security metrics. Each security criterion is specifically designed to highlight potential vulnerabilities and risks, enabling users to make informed decisions based on clear, actionable insights.

### Interpreting the Results

<Card>
Chainstream clearly indicates the result for each criterion as follows:

- **Pass**: Token passes the security criterion with no significant risks detected.
- **Warning**: Potential risk detected; further investigation recommended.
- **Fail**: Token fails the criterion and poses a significant security risk. Immediate caution advised.
</Card>
