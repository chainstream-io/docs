---
title: 'ウォレット - 送金一覧'
openapi: '/jp/api-reference/openapi-data-jp.yaml GET /v2/wallet/{chain}/{walletAddress}/transfers'

<Note>
ウォレットのトークン転送履歴（受信および送信の両方）を取得します。ページネーションとトークンによるフィルタリングに対応しています。
</Note>

**関連**: [GraphQL 転送](/jp/graphql/examples/transfers) | MCP: `wallet_get_balance_updates`
