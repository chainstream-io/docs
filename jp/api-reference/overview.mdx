## 認証

すべての REST / WebSocket リクエストには、API Key、OAuth、ウォレット認証、または [x402 / MPP](/jp/docs/platform/billing-payments/x402-payments) 支払ヘッダが必要です。詳細は [Authentication](/jp/api-reference/authentication/authenticate)。
