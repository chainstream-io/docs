## 概要

ChainStreamは、AIアプリケーション向けに特別に設計されたインフラストラクチャを提供し、AIエージェントがオンチェーンデータの直接クエリ、市場情報の分析、さらには取引操作の実行を可能にします。

## なぜAIにオンチェーンデータが必要なのか

従来のAIモデル（ChatGPT、Claudeなど）にはナレッジカットオフがあり、リアルタイムのオンチェーン情報にアクセスできません。ChainStream AIインフラストラクチャはこれを解決します：

<AccordionGroup>
  <Accordion title="リアルタイムデータアクセス" icon="bolt">
    AIは最新の情報にアクセスできます：
    - トークン価格と取引量
    - ウォレット残高と保有資産
    - DEXの流動性と取引
    - 市場ランキング
  </Accordion>
  
  <Accordion title="オンチェーン分析" icon="chart-mixed">
    AIは以下の分析を実行できます：
    - トークンセキュリティチェック
    - ウォレットPnL計算
    - Smart Moneyトラッキング
    - リスク評価
  </Accordion>
  
  <Accordion title="取引アシスタンス" icon="right-left">

    - 最適な取引ルートのクエリ
    - ガス代の見積もり
    - トランザクションの構築（ユーザー確認が必要）
  </Accordion>

## MCP（Model Context Protocol）

MCPは、AIモデルが外部データソースやツールと対話するためのオープンプロトコルです。ChainStreamは主要なAIモデルをサポートするMCP Serverの実装を提供しています。

### 対応モデル

| モデル | サポート状況 | 備考 |
|-------|------------|------|
| Claude | ネイティブサポート | Anthropic公式MCPサポート |
| GPT-4 | Function Calling経由 | OpenAI function calling |
| Cursor AI | ネイティブサポート | IDE内蔵MCP |
| その他 | 対応中 | 詳細はお問い合わせください |

    - リアルタイム価格と過去のローソク足をクエリ

    - トークン価値の計算
    - 保有PnLの分析
    - ポートフォリオレポートの生成
  </Tab>
