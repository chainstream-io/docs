ChainStream が AI Agent に公開する [データプロダクト](/jp/docs/data-products/overview) と実行プリミティブは、他のクライアントに公開するものと同一です。Agent に必要な認証、ツール探索、従量課金のパターンを加えただけです。

<Note>
  **AI Agents は消費面であり、独立した製品ではありません。** Agent ができることはすべて、
  REST、GraphQL、WebSocket、Kafka、SDK、CLI 経由でも実現できます。
</Note>

## 2 つの主要サーフェス

<CardGroup cols={2}>
  <Card title="MCP Server" icon="robot" href="/jp/docs/ai-agents/mcp-server/introduction">
    ホスト型 (セルフホスト可) の Model Context Protocol サーバ。
    Claude、Cursor、ChatGPT、Windsurf などの MCP 対応クライアントに即接続。
  </Card>
  <Card title="Agent Skills" icon="brain" href="/jp/docs/ai-agents/agent-skills/introduction">
    ChainStream の使い方を Agent に教えるインストール可能な SKILL.md パッケージ。
    トークン分析、Smart Money 追跡、DeFi 実行などのパターン。
  </Card>
</CardGroup>

## どれを選ぶか

| ケース | サーフェス |
|:--|:--|
| Claude / Cursor / ChatGPT で ChainStream ツールを使う | [MCP Server](/jp/docs/ai-agents/mcp-server/introduction) |
| 既製の Skill をインストールしてトークン調査したい | [Agent Skills](/jp/docs/ai-agents/agent-skills/chainstream-data) |
| Python / TS で Agent を自作する | [SDK](/jp/sdks/overview) |
| アカウントなしで USDC 従量課金したい | [x402 支払い](/jp/docs/platform/billing-payments/x402-payments) |

## Agent 向け Quickstart

<Steps>
  <Step title="MCP Server を起動">
    ```bash
    CHAINSTREAM_API_KEY=your_api_key npx @chainstream-io/mcp
    ```
  </Step>
  <Step title="Agent を接続">
    Claude Desktop、Cursor、ChatGPT の設定例は
    [MCP Server セットアップ](/jp/docs/ai-agents/mcp-server/setup)。
  </Step>
  <Step title="質問する">
    *「過去 24 時間の Solana 出来高トップ 10 トークンを、KYT アラートがあるものを強調して表示して」*
  </Step>
</Steps>
