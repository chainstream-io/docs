ChainStream が AI Agent に公開する [データプロダクト](/jp/docs/data-products/overview) と実行プリミティブは、他のクライアントに公開するものと同一です。Agent に必要な認証、ツール探索、従量課金のパターンを加えただけです。

<Note>
  **AI Agents は消費面であり、独立した製品ではありません。** Agent ができることはすべて、
  REST、GraphQL、WebSocket、Kafka、SDK、CLI 経由でも実現できます。
</Note>

## 2 つの主要サーフェス

<CardGroup cols={2}>
  <Card title="MCP Server" icon="robot" href="/jp/docs/ai-agents/mcp-server/introduction">
    ホスト型 (セルフホスト可) の Model Context Protocol サーバ。
    Claude、Cursor、ChatGPT、Windsurf などの MCP 対応クライアントに即接続。
  </Card>
  <Card title="Agent Skills" icon="brain" href="/jp/docs/ai-agents/agent-skills/introduction">
    ChainStream の使い方を Agent に教えるインストール可能な SKILL.md パッケージ。
    3 つのスキルで標準データ（`chainstream-data`）、カスタム GraphQL 分析
    （`chainstream-graphql`）、DeFi 実行（`chainstream-defi`）をカバーします。
    Cursor、Claude Code、Codex 向けに即導入可能。
  </Card>
</CardGroup>

## なぜ Agent にとって重要か

従来の LLM がブロックチェーンにアクセスするとき、3 つの壁に必ず当たります。モデルには知識のカットオフがある、認証済みデータソースを持たない、事前に登録したアカウントなしでは読み取りの支払いができない、という壁です。ChainStream はそれぞれに解を持ちます：

<AccordionGroup>
  <Accordion title="ツール利用向けに型付けされたライブデータ" icon="bolt">
    すべてのデータプロダクトは JSON スキーマを持つ構造化ツールコールとして利用できます。Agent は画面スクレイピングされたテキストではなく、正規化された ID を受け取ります。結果として、毎ステップで再パースせずに呼び出しを連鎖できます。
  </Accordion>
  <Accordion title="Agent ループに合う認証アクセス" icon="shield-halved">
    API Key も OAuth も使えます。自律型 Agent 向けには [x402 / MPP](/jp/docs/platform/billing-payments/x402-payments) を用意しています。HTTP レベルの従量課金スキームで、Agent は USDC で支払って即座に呼び出せます。事前のアカウント登録は不要です。
  </Accordion>
  <Accordion title="ユーザーレベルのガードレール付きオプショナル書き込み" icon="right-left">
    DEX の見積・ルーティング・スワップ送出はすべて利用可能です。書き込みは依然としてユーザー所有のウォレットによる署名済みトランザクションを要求するため、Agent はトランザクションを組み立てられても、他人の資金から無断で送ることはできません。
  </Accordion>
</AccordionGroup>

## どれを選ぶか

| ケース | サーフェス |
|:--|:--|
| Claude / Cursor / ChatGPT で ChainStream ツールを使う | [MCP Server](/jp/docs/ai-agents/mcp-server/introduction) |
| 既製の Agent Skill をインストールしてトークン調査したい | [chainstream-data スキル](/jp/docs/ai-agents/agent-skills/chainstream-data) |
| キューブ横断 JOIN、カスタム集計、時系列が必要 | [chainstream-graphql スキル](/jp/docs/ai-agents/agent-skills/chainstream-graphql) |
| Agent にスワップ／トークン作成を実行させたい | [chainstream-defi スキル](/jp/docs/ai-agents/agent-skills/chainstream-defi) |
| Python / TS で Agent を自作して直接アクセスしたい | [SDK](/jp/sdks/overview) |
| アカウントなしで USDC 従量課金したい | [x402 ペイメント](/jp/docs/platform/billing-payments/x402-payments) |
| Tempo / Stripe / ステーブルコインで支払いたい | [MPP ペイメント](/jp/docs/platform/billing-payments/mpp-payments) |
| Agent がオーケストレーションできるシェルツールが欲しい | [CLI](/jp/docs/access-methods/cli) |

## Agent 向け Quickstart

<Steps>
  <Step title="MCP Server を起動">
    ```bash
    CHAINSTREAM_API_KEY=your_api_key npx @chainstream-io/mcp
    ```
  </Step>
  <Step title="Agent を接続">
    Claude Desktop、Cursor、ChatGPT の設定例は
    [MCP Server セットアップ](/jp/docs/ai-agents/mcp-server/setup)。
  </Step>
  <Step title="質問する">
    *「過去 24 時間の Solana 出来高トップ 10 トークンを、KYT アラートがあるものを強調して表示して」*
  </Step>
</Steps>
