---
title: "コンプライアンス (KYT / KYA)"
description: "データと同じ API に組み込まれた Know-Your-Transaction / Know-Your-Address。"
icon: "shield-halved"
---

コンプライアンスチェックはデータ本体と同じ ChainStream API に組み込まれています。別ベンダー、別請求、別のデータ流出面はいりません。

<CardGroup cols={2}>
  <Card title="KYT — Know Your Transaction" icon="receipt" href="/jp/docs/compliance/kyt-concepts">
    入出金のリスクスコアリング。リスク帯、エクスポージャ分類、
    アラートコード、判断根拠を返します。
  </Card>
  <Card title="KYA — Know Your Address" icon="user-shield" href="/jp/docs/compliance/kya-concepts">
    対向アドレスのフィンガープリンティング。エンティティタグ、
    リスクレベル、制裁マッチ、履歴行動。
  </Card>
</CardGroup>

## 必要になる場面

<AccordionGroup>
  <Accordion title="取引所の入金スクリーニング">
    ユーザーへ反映する前に、入金ごとに KYT を実行。制裁、ミキサー、
    ダークネット市場に該当すればブロック/保留。
  </Accordion>
  <Accordion title="取引所の出金スクリーニング">
    出金先に対して不正評価スコアとネットワーク帰属分解を取得し、
    送金前に確認。
  </Accordion>
  <Accordion title="Travel Rule 報告">
    転送ごとのサマリを、FATF Travel Rule メッセージに必要な
    フィールドで出力。
  </Accordion>
</AccordionGroup>
