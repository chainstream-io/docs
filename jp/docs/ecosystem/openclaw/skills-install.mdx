## 概要

ChainStreamをOpenClawエージェントに接続するには3つの方法があります。ワークフローに最適な方法を選択してください。

| 方法 | 最適な用途 | セットアップ時間 |
|--------|----------|------------|
| [Vercel Skills CLI](#method-1-vercel-skills-cli) | クロスエージェント互換、40以上のエージェントに対応 | 約30秒 |
| [ClawHub](#method-2-clawhub) | OpenClawネイティブ、コミュニティエコシステム | 約1分 |
| [MCP直接設定](#method-3-direct-mcp-config) | 完全な制御、カスタムデプロイメント | 約2分 |

## Method 1: Vercel Skills CLI

Vercel Skills CLIは、対応するあらゆるAIエージェントに1つのコマンドでChainStreamスキルをインストールできるクロスエージェントパッケージマネージャーです。

### クイックインストール

```bash
npx skills add chainstream-io/skills
```

エージェント環境を自動検出し、ChainStream MCPツールを適切に設定します。

### エージェント指定インストール

特定のエージェントを対象にする場合：

```bash
npx skills add chainstream-io/skills -a openclaw
```

### 対応エージェント

Vercel Skills CLIは**40以上のエージェント**に対応しています：

| エージェント | インストールコマンド |
|-------|----------------|
| OpenClaw | `npx skills add chainstream-io/skills -a openclaw` |
| Claude Code | `npx skills add chainstream-io/skills -a claude-code` |
| Cursor | `npx skills add chainstream-io/skills -a cursor` |
| Cline | `npx skills add chainstream-io/skills -a cline` |
| GitHub Copilot | `npx skills add chainstream-io/skills -a copilot` |
| Codex | `npx skills add chainstream-io/skills -a codex` |

完全な互換性テーブルと設定ファイルの場所については、[クロスエージェント互換性](/jp/guides/integrations/openclaw/cross-agent-compatibility)をご覧ください。

<Tip>
  インストール後、APIキーを環境変数として設定してください：

  ```bash
  export CHAINSTREAM_API_KEY=your-key
  ```
</Tip>

## Method 2: ClawHub

[ClawHub](https://clawhub.ai)はOpenClawのネイティブスキルマーケットプレイスです。ChainStreamスキルはコミュニティパッケージとして公開されています。

### スキル詳細

| フィールド | 値 |
|-------|-------|
| URL | [clawhub.ai/chainstream/chainstream-data](https://clawhub.ai/chainstream/chainstream-data) |
| バージョン | 3.1.1 |
| MCPツール | 17 |
| ライセンス | MIT-0 |

### CLIからインストール

```bash
openclaw skills install chainstream-data
```

### ClawHub UIからインストール

1. [clawhub.ai/chainstream/chainstream-data](https://clawhub.ai/chainstream/chainstream-data)にアクセス
2. **Install**をクリック
3. 画面の指示に従ってAPIキーを設定

### APIキーの設定

ClawHubからインストールした後、OpenClawエージェントの設定でAPIキーを指定します：

```yaml
skills:
  chainstream-data:
    env:
      CHAINSTREAM_API_KEY: your-key
```

<Note>
  ClawHubスキルは17のMCPツール（15のコアツールに加え、ClawHub固有のユーティリティ2つ）を公開します。コアツールセットの詳細は[ツールリファレンス](/jp/guides/ai-infrastructure/mcp-server/overview)をご覧ください。
</Note>

## Method 3: MCP直接設定

OpenClawエージェントの設定ファイルでChainStream MCPサーバーを直接設定します。ChainStreamは2つの接続モードを提供しています：

### クラウドエンドポイント（推奨）

ホスト型MCPエンドポイントに接続 — インストール不要、常に最新版：

```yaml
skills:
  chainstream:
    type: mcp
    url: https://mcp.chainstream.io/mcp
    env:
      CHAINSTREAM_API_KEY: your-key
```

### ローカルnpmパッケージ

`@chainstream-io/mcp` npmパッケージを使用してMCPサーバーをローカルで実行します。エージェントフレームワークがローカルのstdioプロセスを必要とする場合に便利です：

```yaml
skills:
  chainstream:
    type: mcp
    command: npx
    args:
      - "@chainstream-io/mcp"
    env:
      CHAINSTREAM_API_KEY: your-key
```

Claude Desktop、Cursor、その他のクライアントでの詳細な設定については、[MCPサーバーセットアップ](/jp/guides/ai-infrastructure/mcp-server/setup-guide)をご覧ください。

## インストールの確認

いずれの方法でインストールした後、OpenClawエージェントに以下のように尋ねてツールが利用可能であることを確認してください：

```
What ChainStream tools do you have access to?
```

エージェントが`tokens_search`、`tokens_analyze`、`wallets_profile`などの利用可能なツールをリストするはずです。
