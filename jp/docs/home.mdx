## ChainStream が提供するもの

### 統一されたマルチチェーンデータ

対応する全チェーンに対する単一スキーマ。トークン価格、DEX トレード、ウォレット PnL、プール流動性、ホルダー分布、OHLC ローソク足、Smart-Money の活動、ブロックチェーンのコアデータ — すべて同じ認証、同じ ID、同じディメンション フィルタで取得できます。

### スタックに合ったリアルタイム配信

<CardGroup cols={3}>
  <Card title="REST + GraphQL" icon="bolt">
    リクエスト/レスポンス。ダッシュボード、ユーザー向けルックアップ、アドホック分析に最適。
  </Card>
  <Card title="WebSocket" icon="tower-broadcast">
    ブラウザ、モバイル、リアルタイム UI 更新が必要なバックエンドへのサブ秒プッシュ。
  </Card>
  <Card title="Kafka Streams" icon="bars-staggered">
    トレーディングエンジン、インデクサ、アラートパイプラインなど遅延センシティブなバックエンド向けの exactly-once 配信。
  </Card>
</CardGroup>

### 内蔵コンプライアンス

KYT（Know Your Transaction）と KYA（Know Your Address）がデータ本体と同じ API 内に存在します。転送をスコアリングし、カウンターパーティを識別し、Travel Rule 形式のレポートを生成する — 2 つ目のベンダー・2 つ目の請求・2 つ目のデータ漏洩経路を持つ必要はありません。

### AI エージェントのファーストクラスサポート

AI エージェントは他のクライアントと同じインターフェースで ChainStream に接続できます。ホスト型 MCP Server が型付きツールとしてカタログを公開し、Agent Skills はインストール可能な SKILL.md パックとして配布。エージェント フレンドリーな認証（API キー、OAuth、x402/MPP マイクロペイメント）により、事前アカウント登録なしでペイパークエリが可能です。
