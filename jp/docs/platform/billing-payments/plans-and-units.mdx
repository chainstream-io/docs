## 概要

ChainStream は**ユニットベースの課金モデル**を採用しています。API によって消費するユニット数が異なり、ビジネスニーズに合ったプランを選択できます。

## ユニット計算

### ユニットとは

ユニットは ChainStream の課金単位です。API によって消費するユニット数が異なり、より複雑な API はより多くのユニットを消費します。

### Data API 消費量

ユニットコストは計算の複雑さに応じて階層化されています。シンプルな検索は低コスト、集計が多いエンドポイントやオンチェーン実行エンドポイントは高コストになります。

## WebSocket リアルタイムサブスクリプション課金

WebSocket リアルタイムデータサブスクリプションは別途課金され、データバイト数に基づいて請求されます：

| 項目 | レート | 説明 |
|:--|:--|:--|
| WebSocket データ | **0.005 ユニット/バイト** | プッシュデータのバイト数で課金 |
| WebSocket 接続 | 無料 | 接続確立には課金なし |
| ハートビートメッセージ | 無料 | ハートビートパケットは課金対象外 |

### WebSocket 使用量見積もり例

```
シナリオ: 10 トークンのリアルタイムローソク足（1 分足）を監視

計算:
- ローソク足メッセージ約 200 バイト/本
- 各トークン約 1 本/分
- 10 トークン × 1 メッセージ/分 × 60 分 × 24 時間 × 30 日 = 432,000 メッセージ
- 合計データ: 432,000 × 200 = 86,400,000 バイト
- 月間: 86,400,000 × 0.005 = 432,000 ユニット

シナリオ: 50 トークンの統計を監視（リアルタイム、約 1 更新/秒）

計算:
- トークン統計メッセージ約 500 バイト/件
- 50 トークン × 1 メッセージ/秒 × 86,400 秒 × 30 日 = 129,600,000 メッセージ
- 合計データ: 129,600,000 × 500 = 64,800,000,000 バイト（約 64.8 GB）
- 月間: 64,800,000,000 × 0.005 = 324,000,000 ユニット（約 324M）
```

<Note>
バッチ API には通常割引があります。正確な消費量については API リファレンスを参照してください。WebSocket メッセージの頻度は市場の活動量に依存し、高ボラティリティ期間にはメッセージ量が増加します。
</Note>

## クォータ管理

### Dashboard での使用量確認

<Steps>
  <Step title="Dashboard にログイン">
    [ChainStream Dashboard](https://www.chainstream.io/dashboard) にアクセス
  </Step>
  <Step title="使用量の確認">
    Usage ページで当月の使用量、残りクォータ、過去のトレンドを確認
  </Step>
</Steps>

### クォータアラート設定

使用量がしきい値に達した際に通知を受け取るクォータアラートを設定できます：

| アラートレベル | トリガー条件 | 通知方法 |
|:--|:--|:--|
| リマインダー | 使用量 50% | メール |
| 警告 | 使用量 80% | メール + Dashboard アラート |
| 重大 | 使用量 100% | メール + Dashboard アラート |

<Tip>
予期しないサービス中断を避けるため、少なくとも 80% アラートを有効にすることを推奨します。
</Tip>

## クォータ消費時の対応

### クォータがなくなるとどうなるか

月間クォータが消費されると、API は 403 エラーを返し、呼び出しを続行できなくなります。有料プランのユーザーは手動で追加クォータを購入してサービスを復旧できます。

| プラン | 対応方法 |
|:--|:--|
| Free | API が 403 エラーを返し、翌月のクォータリセットを待つ |
| Starter / Pro / Enterprise | API が 403 エラーを返し、追加クォータの購入でサービス復旧可能 |
| Custom | 契約条件に従う |

### 追加クォータの購入

有料プランのユーザーは Dashboard で追加クォータを購入でき、プランの超過レートに基づいて料金が設定されます：

| プラン | 追加クォータ料金 |
|:--|:--|
| Starter | $12.9 / 1M ユニット |
| Pro | $9.9 / 1M ユニット |
| Enterprise | $6.9 / 1M ユニット |

<Warning>
クォータ消費時はサービスが即座に中断されます（403 エラー）。80% 使用量アラートを設定し、ビジネスへの影響を避けるために事前に追加クォータを購入することを推奨します。
</Warning>

## AI エージェント決済（x402 / MPP）

Dashboard サインアップなしでプログラマティックアクセスが必要な AI エージェント向けに、ChainStream は 2 つのオンチェーン決済プロトコルをサポートしています：

### x402 (Base / Solana USDC)

[x402 プロトコル](https://www.x402.org/)は HTTP ネイティブのマイクロペイメントを実現します。API 呼び出しが 402 を返すと、エージェントのウォレットが USDC 送金に署名してリトライします — アカウント登録は不要です。

| プロパティ | 値 |
|----------|-------|
| ネットワーク | Base (チェーン ID 8453)、Solana |
| 通貨 | USDC |
| 標準 | EIP-3009 (Base)、ネイティブ送金 (Solana) |
| ガス手数料 | **無料** — ChainStream のファシリテーターがすべてのガスコストを負担 |

<Tip>
支払うのはサブスクリプション料金（USDC）のみです。ChainStream は独自の x402 ファシリテーターを運営しており、オンチェーントランザクションの送信とガス手数料の支払いを代行します。エージェントウォレットにはガス用の ETH や SOL は不要です。
</Tip>

### MPP (Tempo USDC.e)

MPP は Tempo ネットワークを使用して決済を行います。ガスは USDC.e で直接支払われ、ETH は不要です。

| プロパティ | 値 |
|----------|-------|
| ネットワーク | Tempo (チェーン ID 4217) |
| 通貨 | USDC.e |

### 仕組み

1. エージェントが ChainStream API を呼び出し
2. サーバーが 402 を返す（アクティブなサブスクリプションなし）
3. エージェントが `/x402/pricing` から利用可能なプランを取得
4. エージェントが x402 または MPP でプランを購入
5. サーバーが **API Key** を返す — エージェントはこれを保存して今後のすべての呼び出しに使用

## FAQ

<AccordionGroup>
  <Accordion title="最適なプランの選び方は？">
    まず Free プランで実際の使用量を評価し、月間消費量に基づいて選択することを推奨します：
    - 30K ユニット未満: Free
    - 30K〜10M ユニット: Starter
    - 10M〜50M ユニット: Pro
    - 50M ユニット超: Enterprise
  </Accordion>
  <Accordion title="いつでもアップグレード/ダウングレードできますか？">
    はい。アップグレードは即座に適用され、日割り計算で請求されます。ダウングレードは次の請求サイクルから適用されます。
  </Accordion>
  <Accordion title="未使用のクォータは繰り越されますか？">
    いいえ。月間クォータは月末にリセットされ、繰り越されません。
  </Accordion>
  <Accordion title="請求書を取得するには？">
    Dashboard → Billing → Invoices から過去の請求書をダウンロードできます。Enterprise のお客様はセールスにカスタム請求書を依頼できます。
  </Accordion>
</AccordionGroup>
