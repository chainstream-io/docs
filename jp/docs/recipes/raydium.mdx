RaydiumはSolana上最大のAMMおよび流動性プロバイダーです。ChainStreamはREST API、GraphQL、WebSocket、MCPツールを通じて、Raydiumのトレードとプールの完全なデータカバレッジを提供します。

<Info>
**対応オペレーション**: トレード分析、流動性プールモニタリング、トークン価格追跡、OHLCVデータ、ホルダー分析、Raydiumローンチパッドによるトークン作成。
</Info>

## Raydiumの最近のトレードを取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?limit=20" \
  -H "X-API-KEY: your_api_key"
```

パラメータの詳細は[Trade - List](/jp/api-reference/endpoint/data/trade/v2/trade-chain-get)を参照してください。
  </Tab>
  <Tab title="GraphQL">

`DEXTrades`をクエリし、Raydiumのプロトコル名でフィルタリングします：

```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    dexProgram: {is: "RAYDIUM_PROGRAM_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
      }
      Dex { ProgramAddress ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", limit: 20 })
```
  </Tab>
</Tabs>

## Raydium上のトークン価格を取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

詳細は[Token - Price](/jp/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)を参照してください。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

## Raydiumでトークンを作成するには？

ChainStreamはAPI経由でRaydiumローンチパッドでのトークン作成をサポートしています。

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "raydium",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

詳細は[DEX - Create](/jp/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)を参照してください。
  </Tab>
</Tabs>

<Note>
トークン作成はSolanaでのみ利用可能です。秘密鍵は作成トランザクションの署名に使用されます。
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="Pump.funとRaydiumプールの違いは何ですか？">
    Pump.funトークンは卒業するまでボンディングカーブで取引されます。卒業すると、流動性はRaydium AMMプールに移行されます。最近卒業したトークンを見つけるには`ranking/sol/migrated`を使用してください。
  </Accordion>
  <Accordion title="DEXでトレードをフィルタリングするには？">
    GraphQLでは、`dexProgram`セレクターを使用してRaydiumのプログラムアドレスでフィルタリングします。REST APIでは、全DEXのトレードが返されます — レスポンスの`dex`フィールドでRaydiumのトレードを識別できます。
  </Accordion>
  <Accordion title="Raydiumのプールイベントをリアルタイムで追跡できますか？">
    はい。WebSocketサブスクリプションを使用して、リアルタイムのプールおよびトレード更新を受信できます。設定方法は[リアルタイムストリーミング](/jp/docs/access-methods/websocket)を参照してください。
  </Accordion>
</AccordionGroup>
