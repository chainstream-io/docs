ChainStream は現在、データおよび DeFi 操作に対して **3 つのブロックチェーン**をサポートしています。

## チェーン概要

| チェーン | 識別子 | タイプ | ネイティブトークン | Data API | DeFi (Swap) | WebSocket | KYT/KYA |
|-------|------------|------|-------------|----------|-------------|-----------|---------|
| Solana | `sol` | L1 | SOL | Yes | Yes | Yes | Yes |
| BSC | `bsc` | L1 | BNB | Yes | Yes | Yes | Yes |
| Ethereum | `eth` | L1 | ETH | Yes | Yes | Yes | Yes |

<Note>
チェーン識別子は常に**小文字**を使用してください: `sol`、`eth`、`bsc`。
</Note>

## チェーン別 DEX カバレッジ

| チェーン | 対応 DEX |
|-------|-----------------|
| Solana | Jupiter, Raydium, PumpFun, Moonshot, Candy |
| BSC | KyberSwap |
| Ethereum | KyberSwap |

<Note>
トークン作成は Solana のローンチパッドのみで利用可能です: PumpFun、Raydium。
</Note>

## データの利用可能性

### リアルタイムデータ

3 つのチェーンすべてで以下を提供：

- トークンメタデータ、統計、マーケットデータ
- ウォレット残高、PnL、純資産
- 取引履歴とアクティビティフィード
- トークンランキング（ホット、新規、移行済み、卒業間近）
- 保有者分析と分布
- セキュリティ監査フラグ

### WebSocket ストリーミング

すべてのチェーンでリアルタイムサブスクリプションが利用可能：

- トークン価格と統計の更新
- OHLCV ローソク足フィード
- 取引アクティビティストリーム
- ウォレット残高の変動
- 新規トークン作成イベント
- ボンディングカーブの進捗（Solana）

### DeFi 操作

| 操作 | Solana | Ethereum | BSC |
|-----------|--------|----------|-----|
| スワップ（見積もり + 実行） | Yes | Yes | Yes |
| トークン作成（ローンチパッド） | Yes | -- | -- |
| トランザクションブロードキャスト | Yes | Yes | Yes |
| ガス見積もり | -- | Yes | Yes |

## 製品間のチェーン識別子

| コンテキスト | Solana | BSC | Ethereum |
|---------|--------|-----|----------|
| REST API | `sol` | `bsc` | `eth` |
| CLI `--chain` | `sol` | `bsc` | `eth` |
| SDK | `sol` | `bsc` | `eth` |
| MCP Tools | `sol` | `bsc` | `eth` |
| WebSocket | `sol` | `bsc` | `eth` |
