## ChainStream とは

ChainStream は Web3 時代のために構築された AI ネイティブインフラストラクチャです。従来の Web3 データ API とは異なり、ChainStream は初日から AI エージェントと自動化シナリオを想定して設計されており、以下を提供します：

- **統合データレイヤー**: 1 つの API でマルチチェーンデータにアクセス可能。複数のベンダーとの統合は不要
- **組み込みコンプライアンス**: ネイティブ KYT/KYA 統合により、サードパーティのリスク管理サービスが不要
- **AI 実行レイヤー**: MCP プロトコルを通じて AI エージェントがオンチェーンの世界で直接操作可能

<Tip>
開発者と AI エージェントが安全かつ効率的にオンチェーンの世界にアクセスし、操作できるようにします。
</Tip>

## コア機能

ChainStream は 3 つのコア機能を提供し、Web3 データ取得、セキュリティコンプライアンス、AI 実行の完全なチェーンをカバーします：

### データ — リアルタイムオンチェーンデータ

統合 API を通じてマルチチェーンのリアルタイムデータにアクセスし、WebSocket リアルタイムプッシュおよび履歴データクエリをサポートします。

<AccordionGroup>
  <Accordion title="マルチチェーンデータ集約">
    1 つの API エンドポイントで Solana、Ethereum、BSC のデータにアクセスでき、複数のデータソースに個別接続する必要がありません。クロスチェーンクエリとデータ標準化をサポートしています。
  </Accordion>
  <Accordion title="リアルタイム価格・取引データ">
    ミリ秒レベルの価格更新、DEX/CEX 集約価格、取引ペアの板情報、ローソク足データをサポート。WebSocket リアルタイムプッシュは 2ms 未満のレイテンシを実現しています。
  </Accordion>
  <Accordion title="スマートマネーウォレット追跡">
    オンチェーンのスマートマネーの動きを追跡します。ホエールウォレット、機関投資家アドレス、KOL ウォレットなどが含まれます。ウォレットラベル、ポジション変更、取引行動分析を提供します。
  </Accordion>
  <Accordion title="履歴データクエリ">
    任意の期間の履歴データクエリをサポートし、過去の価格、ポジション、取引記録を含みます。データ保持期間はプランによって異なります。
  </Accordion>
</AccordionGroup>

### セキュリティ — セキュリティ＆コンプライアンス

組み込みのエンタープライズグレードのセキュリティコンプライアンス機能で、取引所、ウォレット、DeFi プロトコルのリスク管理ニーズに対応します。

<AccordionGroup>
  <Accordion title="KYT（Know Your Transaction）取引リスク評価">
    リアルタイムの取引リスクレベル分析で、不審な資金フローを特定します。OFAC 制裁リストチェック、ミキサー識別、ダークネット関連検出をサポート。リスクスコア（0-100）と詳細なリスクラベルを返却します。
  </Accordion>
  <Accordion title="KYA（Know Your Address）アドレスプロファイリング">
    包括的なアドレスプロファイリング分析で、アドレスタイプ識別（EOA/コントラクト/マルチシグ）、履歴行動分析、資金ソース追跡、関連アドレスマッピングを含みます。
  </Accordion>
  <Accordion title="コンプライアンス監査サポート">
    規制報告要件に対応する完全な監査ログ。FATF トラベルルール形式のレポート出力をサポートし、主要なコンプライアンスツール（Chainalysis、Elliptic 形式互換）と統合できます。
  </Accordion>
</AccordionGroup>

### AI 実行 — AI 実行レイヤー

MCP（Model Context Protocol）を通じて AI エージェントがブロックチェーンと安全にやり取りできるようにします。

<AccordionGroup>
  <Accordion title="MCP サーバーサポート">
    Anthropic MCP プロトコルをネイティブサポートし、Claude やその他の AI モデルが ChainStream の機能を直接呼び出せます。アダプターコード不要で、すぐに使えます。
  </Accordion>
  <Accordion title="AI エージェントフレンドリーなツールセット">
    AI エージェント向けに設計されたツールインターフェースで、構造化出力、エラー自己修復、コンテキスト保存を含みます。LangChain、AutoGPT、CrewAI などの主要フレームワークをサポートしています。
  </Accordion>
  <Accordion title="自然言語からオンチェーン操作">
    AI が自然言語コマンドでオンチェーンクエリと操作を完了できます。例：「アドレス 0x... の過去 7 日間の取引をクエリ」や「PEPE の価格が $0.00001 を超えたらアラート」。
  </Accordion>
</AccordionGroup>

<Warning>
**セキュリティに関する注意**: AI 実行レイヤーは読み取り操作と事前承認された書き込み操作のみをサポートします。すべての機密操作にはユーザーの確認が必要です。
</Warning>
