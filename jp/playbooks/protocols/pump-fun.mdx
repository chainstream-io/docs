Pump.funはSolana上のリーディングなミームコインローンチパッドです。ChainStreamはREST API、GraphQL、WebSocket、MCP Toolsを通じてPump.funトークンの完全なデータカバレッジを提供しています。

<Info>
**サポートされる操作**：トークン発見、価格追跡、取引分析、ホルダー分析、トークン作成（`pumpfun_create`経由）、卒業/マイグレーション追跡。
</Info>

---

## 新しいPump.funトークンを発見するには？

ランキングエンドポイントを使用して、Pump.funローンチパッドで新しく作成されたトークンを見つけます。

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/ranking/sol/newTokens" \
  -H "X-API-KEY: your_api_key"
```

完全なパラメータリファレンスは[ランキング - 新規トークン](/jp/api-reference/endpoint/data/ranking/v2/ranking-chain-newtokens-get)をご覧ください。
  </Tab>
  <Tab title="GraphQL">

Solana上の`TokenSupplyUpdates`をクエリし、Pump.funトークンでフィルタリング：

```graphql
query {
  TokenSupplyUpdates(
    network: sol
    limit: {count: 20}
    orderBy: Block_Time_DESC
  ) {
    Block { Time }
    TokenSupplyUpdate {
      Currency {
        MintAddress
        Decimals
      }
      PostBalance
      MarketCapInUSD
      PriceInUSD
      TotalSupply
    }
  }
}
```

<Tip>
[GraphQL IDE](https://ide.chainstream.io)を開いて、このクエリをインタラクティブに実行できます。
</Tip>
  </Tab>
  <Tab title="MCP Tool">
```
ranking_get_new_tokens({ chain: "solana" })
```

完全なツールリファレンスは[MCPツールカタログ](/jp/guides/ai-infrastructure/mcp-server/tools-catalog)をご覧ください。
  </Tab>
</Tabs>

## Pump.funトークンの価格を取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

完全なリファレンスは[トークン - 価格](/jp/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)をご覧ください。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

<Note>
`TOKEN_ADDRESS`を実際のPump.funトークンミントアドレスに置き換えてください。
</Note>

## Pump.funトークンの取引を取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?tokenAddress=TOKEN_ADDRESS&limit=20" \
  -H "X-API-KEY: your_api_key"
```

完全なリファレンスは[取引 - リスト](/jp/api-reference/endpoint/data/trade/v2/trade-chain-get)をご覧ください。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
        Account { Owner }
      }
      Dex { ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", tokenAddress: "TOKEN_ADDRESS", limit: 20 })
```
  </Tab>
</Tabs>

## 卒業間近のPump.funトークンを追跡するには？

```bash
  -H "X-API-KEY: your_api_key"
```

完全なリファレンスは[ランキング - ファイナルストレッチ](/jp/api-reference/endpoint/data/ranking/v2/ranking-chain-finalstretch-get)をご覧ください。

```bash
  -H "X-API-KEY: your_api_key"
```

完全なリファレンスは[取引 - トップトレーダー](/jp/api-reference/endpoint/data/trade/v2/trade-chain-top-traders-get)をご覧ください。

```graphql
  DEXTrades(
    network: sol
    limit: {count: 100}

  ) {
    Trade {
      Buy {
        Account { Owner }
        Amount
        PriceInUSD

    sum(of: Trade_Buy_Amount)
  }
}
```
  </Tab>

## Pump.funでトークンを作成するには？

ChainStreamはAPI経由でPump.funでのトークン作成をサポートしています。

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "pumpfun",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

完全なリファレンスは[DEX - 作成](/jp/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)をご覧ください。
  </Tab>
  <Tab title="MCP Tool">
```
pumpfun_create({
  name: "My Token",
  symbol: "MYTKN",
  description: "A test token"
})
```
  </Tab>
</Tabs>

<Note>
トークン作成には資金が入ったSolanaウォレットが必要です。秘密鍵は作成トランザクションの署名に使用されます。
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="Pump.funの卒業とは？">
    Pump.funトークンのボンディングカーブが100%に達すると「卒業」し、流動性がRaydiumプールにマイグレーションされ、オープンマーケットで取引が開始されます。卒業間近のトークンを追跡するには`ranking/sol/finalStretch`エンドポイントを、すでに卒業したトークンには`ranking/sol/migrated`を使用してください。
  </Accordion>
  <Accordion title="ボット/MEV取引をフィルタリングするには？">
    GraphQLの`DEXTrades`キューブにはデフォルトフィルター`IsSuspect = false`があり、疑わしいボットやMEV取引を自動的に除外します。REST APIの場合、トレードレスポンスの`isSuspect`フィールドを確認してください。
  </Accordion>
  <Accordion title="Pump.funトークンをリアルタイムで追跡できますか？">
    はい。WebSocket購読を使用してリアルタイムの価格更新と取引ストリームを受信できます。セットアップ手順は[リアルタイムストリーミング](/jp/guides/data-concepts/realtime-streaming)をご覧ください。
  </Accordion>
  <Accordion title="どのチェーンがPump.funをサポートしていますか？">
    Pump.funはSolana専用のローンチパッドです。すべてのPump.funクエリで`chain: sol`（REST）または`network: sol`（GraphQL）を使用してください。
  </Accordion>
</AccordionGroup>
