RaydiumはSolana上で最大のAMMおよび流動性プロバイダーです。ChainStreamはREST API、GraphQL、WebSocket、MCP Toolsを通じてRaydiumの取引とプールの完全なデータカバレッジを提供しています。

<Info>
**サポートされる操作**：取引分析、流動性プール監視、トークン価格追跡、OHLCVデータ、ホルダー分析、Raydiumローンチパッドによるトークン作成。
</Info>

---

## Raydiumの最近の取引を取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?limit=20" \
  -H "X-API-KEY: your_api_key"
```

完全なパラメータリファレンスは[取引 - リスト](/jp/api-reference/endpoint/data/trade/v2/trade-chain-get)をご覧ください。
  </Tab>
  <Tab title="GraphQL">

`DEXTrades`をクエリし、Raydiumのプロトコル名でフィルタリング：

```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    dexProgram: {is: "RAYDIUM_PROGRAM_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
      }
      Dex { ProgramAddress ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", limit: 20 })
```
  </Tab>
</Tabs>

## Raydiumでのトークン価格を取得するには？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

完全なリファレンスは[トークン - 価格](/jp/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)をご覧ください。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

```bash
  -H "X-API-KEY: your_api_key"
```

```graphql

  DEXPools(
    network: sol
    limit: {count: 10}

      Dex { ProgramAddress }
      Base { Address PostAmount }

```


## Raydiumでトークンを作成するには？

ChainStreamはAPI経由でRaydiumローンチパッドでのトークン作成をサポートしています。

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "raydium",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

完全なリファレンスは[DEX - 作成](/jp/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)をご覧ください。
  </Tab>
</Tabs>

<Note>
トークン作成はSolanaでのみ利用可能です。秘密鍵は作成トランザクションの署名に使用されます。
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="Pump.funプールとRaydiumプールの違いは？">
    Pump.funトークンは卒業するまでボンディングカーブで取引されます。卒業後、流動性はRaydium AMMプールにマイグレーションされます。最近卒業したトークンを見つけるには`ranking/sol/migrated`を使用してください。
  </Accordion>
  <Accordion title="DEXで取引をフィルタリングするには？">
    GraphQLでは、`dexProgram`セレクターを使用してRaydiumのプログラムアドレスでフィルタリングします。RESTでは、取引はすべてのDEXにわたって返されます — レスポンスの`dex`フィールドを確認してRaydium取引を識別してください。
  </Accordion>
  <Accordion title="Raydiumプールイベントをリアルタイムで追跡できますか？">
    はい。WebSocket購読を使用してリアルタイムのプールおよび取引更新を受信できます。セットアップ手順は[リアルタイムストリーミング](/jp/guides/data-concepts/realtime-streaming)をご覧ください。
  </Accordion>
</AccordionGroup>
