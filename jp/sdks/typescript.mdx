## REST APIの例

トークン情報の照会：

```typescript
import { ChainStreamClient } from '@chainstream-io/sdk';

const ACCESS_TOKEN = 'YOUR_ACCESS_TOKEN';

async function main() {
  // SDKクライアントを初期化
  const client = new ChainStreamClient(ACCESS_TOKEN);

  const chain = 'sol';
  const tokenAddress = 'So11111111111111111111111111111111111111112'; // SOL

  console.log(`トークン照会: ${chain}/${tokenAddress}`);

  try {
    // SDK経由でAPIを呼び出し
    const result = await client.token.getToken(chain, tokenAddress);
    
    console.log('結果:');
    console.log(JSON.stringify(result, null, 2));
  } catch (error) {
    console.error('API呼び出し失敗:', error);
  }
}

main();
```

## WebSocketの例

リアルタイムのトークンキャンドルデータを購読：

```typescript
import { ChainStreamClient, Resolution } from '@chainstream-io/sdk';

const ACCESS_TOKEN = 'YOUR_ACCESS_TOKEN';

async function main() {
  const client = new ChainStreamClient(ACCESS_TOKEN, {
    autoConnectWebSocket: true,
  });

  const chain = 'sol';
  const tokenAddress = 'So11111111111111111111111111111111111111112'; // SOL

  console.log(`トークンキャンドルを購読: ${chain}/${tokenAddress}`);
  console.log('リッスン中... (Ctrl+Cで停止)\n');

  // SDK経由でWebSocketを購読
  client.stream.subscribeTokenCandles({
    chain,
    tokenAddress,
    resolution: Resolution['1s'],
    callback: data => {
      console.log(new Date().toISOString(), JSON.stringify(data));
    },
  });

  // プロセスを維持
  process.on('SIGINT', () => {
    console.log('\n接続を閉じています...');
    process.exit(0);
  });

  // 実行を継続
  await new Promise(() => {});
}

main().catch(error => {
  console.error('テスト失敗:', error.message);
  process.exit(1);
});
```

## その他の例

### トークン検索

```typescript
const results = await client.token.searchToken('bonk');
console.log(results);
```

完全なパラメータは[トークン - 検索](/jp/api-reference/endpoint/data/token/v2/token-search-get)をご覧ください。

### 取引リストの取得

```typescript
const trades = await client.trade.getTrades('sol', {
  tokenAddress: 'TOKEN_ADDRESS',
  limit: 20,
});
console.log(trades);
```

完全なパラメータは[取引 - リスト](/jp/api-reference/endpoint/data/trade/v2/trade-chain-get)をご覧ください。

### ウォレットPnLの取得

```typescript
const pnl = await client.wallet.getPnl('sol', 'WALLET_ADDRESS');
console.log(`総PnL: ${pnl.totalPnlUsd}`);
```

完全なパラメータは[ウォレット - PnL](/jp/api-reference/endpoint/data/wallet/v2/wallet-chain-walletaddress-pnl-get)をご覧ください。

### トークンキャンドル（K線）の取得

```typescript
const candles = await client.token.getCandles('sol', 'TOKEN_ADDRESS', {
  resolution: '1h',
  limit: 24,
});
for (const c of candles) {
  console.log(`${c.time}: O=${c.open} H=${c.high} L=${c.low} C=${c.close}`);
}
```

完全なパラメータは[トークン - キャンドル](/jp/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-candles-get)をご覧ください。

```typescript
const newTokens = await client.ranking.getNewTokens('sol');
for (const token of newTokens) {
  console.log(`${token.symbol}: ${token.address}`);

```

完全なパラメータは[ランキング - 新規トークン](/jp/api-reference/endpoint/data/ranking/v2/ranking-chain-newtokens-get)をご覧ください。

## エラーハンドリング

```typescript
import { ChainStreamClient, ChainStreamError } from '@chainstream-io/sdk';

try {
  const token = await client.token.getToken('sol', 'INVALID_ADDRESS');
} catch (error) {
  if (error instanceof ChainStreamError) {
    console.error(`エラーコード: ${error.code}`);
    console.error(`エラーメッセージ: ${error.message}`);
  }
}
```

### エクスポネンシャルバックオフによるリトライ

レート制限されたリクエスト（HTTP 429）にはリトライロジックを実装：

```typescript
async function withRetry<T>(fn: () => Promise<T>, maxRetries = 3): Promise<T> {
  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      return await fn();
    } catch (error) {
      if (error instanceof ChainStreamError && error.code === 429 && attempt < maxRetries) {
        const delay = Math.pow(2, attempt) * 1000;
        await new Promise(resolve => setTimeout(resolve, delay));
        continue;
      }
      throw error;
    }
  }
  throw new Error('Max retries exceeded');
}

const token = await withRetry(() => client.token.getToken('sol', 'TOKEN_ADDRESS'));
```
