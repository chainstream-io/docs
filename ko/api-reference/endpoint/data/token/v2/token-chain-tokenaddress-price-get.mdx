<Note>
토큰의 현재 가격을 USD 및 네이티브 통화 단위로 가져옵니다. 과거 가격이 필요하면 `/prices` 엔드포인트 또는 GraphQL `DEXTrades` Cube를 사용하세요.
</Note>

**관련**: [GraphQL DEX 거래](/ko/graphql/examples/dex-trades) | MCP: `token_get_price`
