## 인증

모든 REST / WebSocket 요청에는 API Key, OAuth, 지갑 인증, 또는 [x402 / MPP](/ko/docs/platform/billing-payments/x402-payments) 결제 헤더가 필요합니다. 자세한 내용은 [Authentication](/ko/api-reference/authentication/authenticate).
