## 버전 정책

### API 버전

- **Major** (v1, v2): 호환되지 않는 변경
- **Minor** (v1.1, v1.2): 새 기능, 하위 호환
- **Patch** (v1.1.1): 버그 수정

### SDK 버전

SDK는 Semantic Versioning(SemVer)을 따릅니다:
- **Major**: 호환되지 않는 변경
- **Minor**: 새 기능
- **Patch**: 버그 수정

### 지원 중단 정책

- 지원 중단된 엔드포인트는 최소 6개월간 유지
- 응답 헤더에 지원 중단 경고 추가
- 주요 변경 사항은 30일 전에 공지
