## 개요

ChainStream은 AI 애플리케이션을 위해 특별히 설계된 인프라를 제공하여, AI 에이전트가 온체인 데이터를 직접 조회하고, 시장 정보를 분석하며, 트레이딩 작업까지 실행할 수 있도록 합니다.

## AI에 온체인 데이터가 필요한 이유

기존 AI 모델(ChatGPT, Claude 등)은 학습 데이터 기준일이 있어 실시간 온체인 정보에 접근할 수 없습니다. ChainStream AI 인프라는 이 문제를 해결합니다:

<AccordionGroup>
  <Accordion title="실시간 데이터 접근" icon="bolt">
    AI가 최신 데이터에 접근할 수 있습니다:
    - 토큰 가격 및 거래량
    - 지갑 잔액 및 보유 내역
    - DEX 유동성 및 거래
    - 시장 랭킹
  </Accordion>
  
  <Accordion title="온체인 분석" icon="chart-mixed">
    AI가 수행할 수 있는 분석:
    - 토큰 보안 검사
    - 지갑 PnL 계산
    - Smart Money 추적
    - 리스크 평가
  </Accordion>
  
  <Accordion title="트레이딩 지원" icon="right-left">
    AI가 사용자를 도울 수 있는 영역:
    - 최적 거래 경로 조회
    - 가스 비용 추정
    - 트랜잭션 생성 (사용자 확인 필요)
  </Accordion>
</AccordionGroup>

## MCP (Model Context Protocol)

MCP는 AI 모델이 외부 데이터 소스 및 도구와 상호작용할 수 있게 해주는 개방형 프로토콜입니다. ChainStream은 주요 AI 모델을 지원하는 MCP Server 구현을 제공합니다.

### 지원 모델

| 모델 | 지원 상태 | 비고 |
|-------|-----------|------|
| Claude | 네이티브 지원 | Anthropic 공식 MCP 지원 |
| GPT-4 | Function Calling 방식 | OpenAI function calling |
| Cursor AI | 네이티브 지원 | IDE 내장 MCP |
| 기타 | 준비 중 | 자세한 내용은 문의해 주세요 |
