ChainStream이 AI Agent에게 노출하는 [데이터 제품](/ko/docs/data-products/overview)과 실행 프리미티브는 다른 클라이언트용과 동일합니다. 단지 Agent에 실제로 필요한 인증, 도구 탐색, 종량 과금 패턴이 추가되었을 뿐입니다.

<Note>
  **AI Agents는 소비 표면이며, 별도 제품이 아닙니다.** Agent가 할 수 있는 모든 것은
  REST, GraphQL, WebSocket, Kafka, SDK, CLI를 통해서도 가능합니다.
</Note>

## 두 가지 주요 표면

<CardGroup cols={2}>
  <Card title="MCP Server" icon="robot" href="/ko/docs/ai-agents/mcp-server/introduction">
    호스팅형(자체 호스팅 가능) Model Context Protocol 서버.
    Claude, Cursor, ChatGPT, Windsurf 등 MCP 호환 클라이언트에서 즉시 사용.
  </Card>
  <Card title="Agent Skills" icon="brain" href="/ko/docs/ai-agents/agent-skills/introduction">
    Agent에게 ChainStream 사용법을 가르치는 설치 가능한 SKILL.md 패키지.
    세 개의 스킬이 표준 데이터(`chainstream-data`), 커스텀 GraphQL 분석(`chainstream-graphql`), DeFi 실행(`chainstream-defi`)을 커버합니다. Cursor, Claude Code, Codex에서 바로 사용할 수 있습니다.
  </Card>
</CardGroup>

## Agent에게 이것이 중요한 이유

기존 LLM의 블록체인 접근은 세 가지 지점에서 실패합니다: 모델에는 지식 컷오프가 존재하고, 인증된 데이터 소스가 없으며, 사전에 등록된 계정 없이는 읽기 비용을 지불할 수 없습니다. ChainStream은 이 각각을 해결합니다:

<AccordionGroup>
  <Accordion title="도구 사용에 맞춰 타입화된 실시간 데이터" icon="bolt">
    모든 데이터 제품은 JSON 스키마가 붙은 구조화된 도구 호출로 접근할 수 있습니다.
    Agent는 스크린 스크레이핑된 텍스트가 아닌 정규화된 ID를 받으므로, 매 단계마다
    다시 파싱할 필요 없이 호출을 체이닝할 수 있습니다.
  </Accordion>
  <Accordion title="Agent 루프에 맞는 인증된 접근" icon="shield-halved">
    API 키와 OAuth가 모두 동작합니다. 자율 Agent를 위해서는
    [x402 / MPP](/ko/docs/platform/billing-payments/x402-payments)가 있습니다 —
    호출당 HTTP 결제 스킴이므로 Agent는 USDC로 즉시 결제하고 호출할 수 있으며,
    사전 계정 설정이 필요 없습니다.
  </Accordion>
  <Accordion title="사용자 수준 가드레일이 있는 선택적 쓰기 경로" icon="right-left">
    DEX 견적, 라우팅, 스왑 제출이 모두 가능합니다. 쓰기 작업에는 여전히
    사용자가 제어하는 지갑의 서명된 트랜잭션이 필요합니다 — Agent는 트랜잭션을
    준비할 수는 있지만 다른 사람의 자금을 조용히 전송할 수는 없습니다.
  </Accordion>
</AccordionGroup>

## 어떤 것을 선택할까

| 상황 | 표면 |
|:--|:--|
| Claude / Cursor / ChatGPT에서 ChainStream 도구 사용 | [MCP Server](/ko/docs/ai-agents/mcp-server/introduction) |
| 토큰 리서치용 Skill을 바로 설치 | [chainstream-data 스킬](/ko/docs/ai-agents/agent-skills/chainstream-data) |
| 큐브 간 JOIN, 커스텀 집계, 시계열이 필요 | [chainstream-graphql 스킬](/ko/docs/ai-agents/agent-skills/chainstream-graphql) |
| Agent가 스왑 / 토큰 생성을 실행하길 원함 | [chainstream-defi 스킬](/ko/docs/ai-agents/agent-skills/chainstream-defi) |
| Python/TS로 직접 Agent 구현 | [SDK](/ko/sdks/overview) |
| 계정 없이 USDC 종량 결제 | [x402 결제](/ko/docs/platform/billing-payments/x402-payments) |
| Tempo / Stripe / 스테이블코인으로 결제 | [MPP 결제](/ko/docs/platform/billing-payments/mpp-payments) |
| Agent가 오케스트레이션할 쉘 기반 도구가 필요 | [CLI](/ko/docs/access-methods/cli) |

## Agent용 Quickstart

<Steps>
  <Step title="MCP Server 실행">
    ```bash
    CHAINSTREAM_API_KEY=your_api_key npx @chainstream-io/mcp
    ```
  </Step>
  <Step title="Agent 연결">
    Claude Desktop, Cursor, ChatGPT용 설정 예시는
    [MCP Server 설정](/ko/docs/ai-agents/mcp-server/setup).
  </Step>
  <Step title="질문하기">
    *"지난 24시간 Solana 거래량 Top 10 토큰을 KYT 경고와 함께 알려줘."*
  </Step>
</Steps>
