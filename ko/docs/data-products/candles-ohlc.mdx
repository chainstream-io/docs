**Candles (OHLC)** 데이터 제품은 원래 [Trades](/ko/docs/data-products/trades)에서 직접 집계해야 할 바 (Open / High / Low / Close / Volume)의 사전 집계 버전입니다. 다중 구간으로 토큰 / 페어 / 풀 단위로 제공됩니다.

## 필드

| 필드 | 설명 |
|:--|:--|
| `open`, `high`, `low`, `close` | USD 또는 quote 표시 (설정 가능) |
| `volumeUsd`, `volumeBase`, `volumeQuote` | 방향별 거래량 |
| `trades` | 바 내 거래 수 |
| `buyers`, `sellers` | 고유 주소 수 |
| `timestamp` | 바 시작 시각 (UTC) |

## 구간

`1s` / `1m` / `5m` / `15m` / `1h` / `4h` / `1d` / `1w`
