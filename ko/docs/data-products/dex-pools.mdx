**DEX Pools** 데이터 제품은 ChainStream이 인덱싱하는 모든 유동성 풀의 상태를 제공합니다 — 기존 Uniswap v2 상수곱 풀, v3/v4 집중 유동성 풀, Solana AMM 등 정규화된 필드로 제공됩니다.

## 필드

| 분류 | 예시 |
|:--|:--|
| Identity | `chainId`, `poolAddress`, `dex`, `pairType` |
| Tokens | `token0`, `token1`, `reserve0`, `reserve1` |
| Economics | `tvlUsd`, `volume24hUsd`, `feeTierBps`, `apy7d` |
| State | `priceToken0InToken1`, `sqrtPriceX96`, `tick`, `liquidity` |
