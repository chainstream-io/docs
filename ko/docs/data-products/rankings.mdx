**Rankings** 데이터 제품은 의견이 담긴, 지속적으로 재계산되는 토큰 리스트입니다. 뒤의 로직이 투명하게 공개되어 있으므로 용도에 맞는지 스스로 판단할 수 있습니다.

## 주요 리스트

| 리스트 | 내용 | 갱신 |
|:--|:--|:--|
| `hot` | 최근 1h/24h 고유 구매자·거래량 급증 | 1분 |
| `new` | 최소 유동성/홀더 체크를 통과한 신규 토큰 | 블록 |
| `migrated` | 런치패드에서 완전 DEX로 막 이전 | 1분 |
| `final-stretch` | 이전 임계에 가까운 런치패드 토큰 | 1분 |
| `stock-like` | 주식 유사 프로파일 (저변동·꾸준 축적) | 1시간 |
| `top-gainers` / `top-losers` | 1h/24h/7d 가격 이동 | 5분 |
