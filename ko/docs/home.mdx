## ChainStream이란

ChainStream은 Web3 시대를 위해 구축된 AI 네이티브 인프라입니다. 기존의 Web3 데이터 API와 달리, ChainStream은 처음부터 AI 에이전트와 자동화 시나리오를 위해 설계되었습니다:

- **통합 데이터 레이어**: 하나의 API로 멀티체인 데이터에 접근하며, 여러 벤더를 통합할 필요가 없습니다
- **내장 컴플라이언스**: 네이티브 KYT/KYA 통합으로 서드파티 리스크 제어 서비스가 불필요합니다
- **AI 실행 레이어**: MCP 프로토콜을 통해 AI 에이전트가 온체인 세계에서 직접 작업할 수 있습니다

<Tip>
개발자와 AI 에이전트가 안전하고 효율적으로 온체인 세계에 접근하고 운영할 수 있도록 합니다.
</Tip>

## 핵심 기능

ChainStream은 Web3 데이터 수집, 보안 컴플라이언스, AI 실행의 전체 체인을 포괄하는 세 가지 핵심 기능을 제공합니다:

### 데이터 — 실시간 온체인 데이터

통합 API를 통해 멀티체인 실시간 데이터에 접근하며, WebSocket 실시간 푸시와 히스토리컬 데이터 조회를 지원합니다.

<AccordionGroup>
  <Accordion title="멀티체인 데이터 통합">
    하나의 API 엔드포인트로 Solana, Ethereum, BSC 데이터에 접근하며 여러 데이터 소스에 개별적으로 연결할 필요가 없습니다. 크로스체인 쿼리와 데이터 표준화를 지원합니다.
  </Accordion>
  <Accordion title="실시간 가격 및 거래 데이터">
    밀리초 단위의 가격 업데이트로, DEX/CEX 통합 가격, 거래쌍 뎁스, 캔들스틱 데이터를 지원합니다. WebSocket 실시간 푸시는 &lt;2ms 지연 시간을 제공합니다.
  </Accordion>
  <Accordion title="스마트 머니 지갑 추적">
    온체인 스마트 머니의 움직임을 추적합니다. 고래 지갑, 기관 주소, KOL 지갑 등이 포함됩니다. 지갑 라벨, 포지션 변화, 거래 행동 분석을 제공합니다.
  </Accordion>
  <Accordion title="히스토리컬 데이터 조회">
    모든 기간의 히스토리컬 데이터 조회를 지원하며, 과거 가격, 포지션, 거래 기록이 포함됩니다. 데이터 보존 기간은 플랜에 따라 다릅니다.
  </Accordion>
</AccordionGroup>

### 보안 — 보안 및 컴플라이언스

거래소, 지갑, DeFi 프로토콜의 리스크 제어 요구 사항을 충족하는 엔터프라이즈급 보안 컴플라이언스 기능이 내장되어 있습니다.

<AccordionGroup>
  <Accordion title="KYT (Know Your Transaction) 거래 리스크 평가">
    실시간 거래 리스크 수준 분석으로 의심스러운 자금 흐름을 식별합니다. OFAC 제재 목록 확인, 믹서 식별, 다크넷 연관 탐지를 지원합니다. 리스크 점수(0-100)와 상세 리스크 라벨을 반환합니다.
  </Accordion>
  <Accordion title="KYA (Know Your Address) 주소 프로파일링">
    종합적인 주소 프로파일링 분석으로, 주소 유형 식별(EOA/컨트랙트/멀티시그), 과거 행동 분석, 자금 출처 추적, 연관 주소 매핑이 포함됩니다.
  </Accordion>
  <Accordion title="컴플라이언스 감사 지원">
    규제 보고 요구 사항을 충족하는 완전한 감사 로그를 제공합니다. FATF Travel Rule 형식 보고서 내보내기를 지원하며, 주요 컴플라이언스 도구(Chainalysis, Elliptic 형식 호환)와 통합됩니다.
  </Accordion>
</AccordionGroup>

### AI 실행 — AI 실행 레이어

MCP(Model Context Protocol)를 통해 AI 에이전트가 블록체인과 안전하게 상호작용할 수 있습니다.

<AccordionGroup>
  <Accordion title="MCP 서버 지원">
    Anthropic MCP 프로토콜을 네이티브 지원하여, Claude 및 기타 AI 모델이 ChainStream 기능을 직접 호출할 수 있습니다. 어댑터 코드가 필요 없으며 바로 사용 가능합니다.
  </Accordion>
  <Accordion title="AI 에이전트 친화적 도구 세트">
    AI 에이전트를 위해 특별히 설계된 도구 인터페이스로, 구조화된 출력, 오류 자가 복구, 컨텍스트 보존이 포함됩니다. LangChain, AutoGPT, CrewAI 등 주요 프레임워크를 지원합니다.
  </Accordion>
  <Accordion title="자연어를 온체인 작업으로 변환">
    AI가 자연어 명령을 통해 온체인 조회와 작업을 완료할 수 있습니다. 예: "주소 0x...의 최근 7일 거래 조회" 또는 "PEPE 가격이 $0.00001을 초과하면 알림".
  </Accordion>
</AccordionGroup>

<Warning>
**보안 참고**: AI 실행 레이어는 읽기 작업과 사전 승인된 쓰기 작업만 지원합니다. 모든 민감한 작업에는 사용자 확인이 필요합니다.
</Warning>
