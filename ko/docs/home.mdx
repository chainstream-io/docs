## ChainStream이 제공하는 것

### 통합된 멀티체인 데이터

지원되는 모든 체인에 대한 단일 스키마. 토큰 가격, DEX 거래, 지갑 PnL, 풀 유동성, 홀더 분포, OHLC 캔들, Smart-Money 활동, 블록체인 코어 데이터 — 모두 동일한 인증, 동일한 ID, 동일한 차원 필터로 접근할 수 있습니다.

### 스택에 맞는 실시간 전달

<CardGroup cols={3}>
  <Card title="REST + GraphQL" icon="bolt">
    요청-응답. 대시보드, 사용자 조회, 애드혹 분석에 이상적.
  </Card>
  <Card title="WebSocket" icon="tower-broadcast">
    브라우저, 모바일, 실시간 UI 업데이트가 필요한 백엔드에 서브초 푸시.
  </Card>
  <Card title="Kafka Streams" icon="bars-staggered">
    지연에 민감한 백엔드(트레이딩 엔진, 인덱서, 알림 파이프라인)를 위한 exactly-once 전달.
  </Card>
</CardGroup>

### 내장 컴플라이언스

KYT (Know Your Transaction)와 KYA (Know Your Address)가 데이터 API와 같은 곳에 있습니다. 전송에 점수를 매기고, 상대방의 지문을 식별하고, Travel-Rule 형식 보고서를 생성 — 두 번째 벤더도, 두 번째 청구서도, 두 번째 데이터 유출 표면도 필요 없습니다.

### AI 에이전트 일등 시민 지원

AI 에이전트는 다른 모든 클라이언트와 동일한 방식으로 ChainStream에 접근합니다. 호스팅되는 MCP Server가 카탈로그를 타입드 도구로 노출하고, Agent Skills는 설치 가능한 SKILL.md 팩으로 제공되며, 에이전트 친화적 인증(API 키, OAuth, x402/MPP 마이크로페이먼트)을 통해 에이전트는 사전 계정 설정 없이 건별 과금으로 쿼리할 수 있습니다.
