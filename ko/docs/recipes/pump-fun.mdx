Pump.fun은 Solana의 대표적인 밈코인 런치패드입니다. ChainStream은 REST API, GraphQL, WebSocket, MCP 도구를 통해 Pump.fun 토큰에 대한 완전한 데이터 커버리지를 제공합니다.

<Info>
**지원 작업**: 토큰 발견, 가격 추적, 거래 분석, 홀더 분석, 토큰 생성(`pumpfun_create` 사용), 졸업/이전 추적.
</Info>

---

## 새로운 Pump.fun 토큰을 발견하려면?

랭킹 엔드포인트를 사용하여 Pump.fun 런치패드에서 새로 생성된 토큰을 찾습니다.

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/ranking/sol/newTokens" \
  -H "X-API-KEY: your_api_key"
```

전체 파라미터 참조는 [Ranking - New Tokens](/ko/api-reference/endpoint/data/ranking/v2/ranking-chain-newtokens-get)를 확인하세요.
  </Tab>
  <Tab title="GraphQL">

Solana의 `TokenSupplyUpdates`를 쿼리하고 Pump.fun 토큰으로 필터링합니다:

```graphql
query {
  TokenSupplyUpdates(
    network: sol
    limit: {count: 20}
    orderBy: Block_Time_DESC
  ) {
    Block { Time }
    TokenSupplyUpdate {
      Currency {
        MintAddress
        Decimals
      }
      PostBalance
      MarketCapInUSD
      PriceInUSD
      TotalSupply
    }
  }
}
```

<Tip>
[GraphQL IDE에서 열기](https://ide.chainstream.io)를 통해 이 쿼리를 인터랙티브하게 실행할 수 있습니다.
</Tip>
  </Tab>
  <Tab title="MCP Tool">
```
ranking_get_new_tokens({ chain: "solana" })
```

전체 도구 참조는 [MCP 도구 카탈로그](/ko/guides/ai-infrastructure/mcp-server/tools-catalog)를 확인하세요.
  </Tab>
</Tabs>

## Pump.fun 토큰의 가격을 조회하려면?

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

자세한 내용은 [Token - Price](/ko/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)를 참조하세요.
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

<Note>
`TOKEN_ADDRESS`를 실제 Pump.fun 토큰 민트 주소로 교체하세요.
</Note>

## Pump.fun 토큰의 거래를 조회하려면?

<Tabs>
  <Tab title="REST API">
```bash

  -H "X-API-KEY: your_api_key"
```

자세한 내용은 [Trade - List](/ko/api-reference/endpoint/data/trade/v2/trade-chain-get)를 참조하세요.

```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
        Account { Owner }
      }
      Dex { ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", tokenAddress: "TOKEN_ADDRESS", limit: 20 })
```
  </Tab>
</Tabs>

## Pump.fun에서 토큰을 생성하려면?

ChainStream은 API를 통해 Pump.fun에서의 토큰 생성을 지원합니다.

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "pumpfun",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

자세한 내용은 [DEX - Create](/ko/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)를 참조하세요.
  </Tab>
  <Tab title="MCP Tool">
```
pumpfun_create({
  name: "My Token",
  symbol: "MYTKN",
  description: "A test token"
})
```
  </Tab>
</Tabs>

<Note>
토큰 생성에는 잔액이 있는 Solana 지갑이 필요합니다. 개인 키는 생성 트랜잭션 서명에 사용됩니다.
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="Pump.fun 졸업이란 무엇인가요?">
    Pump.fun 토큰의 본딩 커브가 100%에 도달하면 "졸업"합니다 — 유동성이 Raydium 풀로 이전되고 토큰은 오픈 마켓에서 거래를 시작합니다. 졸업에 가까운 토큰을 추적하려면 `ranking/sol/finalStretch` 엔드포인트를 사용하고, 이미 졸업한 토큰에는 `ranking/sol/migrated`를 사용하세요.
  </Accordion>
  <Accordion title="봇/MEV 거래를 필터링하려면?">
    GraphQL의 `DEXTrades` 큐브에는 기본 필터 `IsSuspect = false`가 있어 봇 및 MEV로 의심되는 거래를 자동으로 제외합니다. REST API에서는 거래 응답의 `isSuspect` 필드를 확인하세요.
  </Accordion>
  <Accordion title="Pump.fun 토큰을 실시간으로 추적할 수 있나요?">
    네. WebSocket 구독을 사용하여 실시간 가격 업데이트 및 거래 스트림을 수신할 수 있습니다. 설정 방법은 [실시간 스트리밍](/ko/guides/data-concepts/realtime-streaming)을 참조하세요.
  </Accordion>
  <Accordion title="Pump.fun은 어떤 체인을 지원하나요?">
    Pump.fun은 Solana 전용 런치패드입니다. 모든 Pump.fun 쿼리에는 `chain: sol`(REST) 또는 `network: sol`(GraphQL)을 사용하세요.
  </Accordion>
</AccordionGroup>
