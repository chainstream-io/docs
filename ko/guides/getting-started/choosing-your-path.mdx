## 데이터 사용자

### 사용 사례

디스플레이, 분석 또는 거래 결정을 위해 온체인 데이터를 얻어야 하는 개발자.

**대표 시나리오:**
- dApp에서 실시간 토큰 가격 표시
- 시장 대시보드 또는 데이터 분석 도구 구축
- 특정 지갑의 포지션 변화 모니터링
- 시장 데이터를 가져오는 트레이딩 봇 개발

**대표 사용자:**
- DeFi 애플리케이션 개발자
- 시장 데이터 표시 플랫폼
- 온체인 데이터 분석가
- 트레이딩 봇 개발자

### 추천 경로

<Steps>
  <Step title="데이터 모델 이해">
    [데이터 개념](/ko/guides/data-concepts/realtime-streaming)을 읽고 ChainStream의 데이터 구조를 이해하세요
  </Step>
  <Step title="토큰 API 호출">
    [토큰 API 레퍼런스](/ko/api-reference/endpoint/token/v1/token-chain-tokenaddress-get)를 참고하여 가격과 토큰 정보를 가져오세요
  </Step>
  <Step title="실습: 가격 알림">
    [가격 알림 봇 튜토리얼](/ko/playbooks/tutorials/build-price-alert-bot)을 따라 완전한 애플리케이션을 구축하세요
  </Step>
</Steps>

### 핵심 기능

다음 기능을 사용하게 됩니다:
- 실시간 가격 데이터 (WebSocket 푸시)
- 멀티체인 토큰 정보 조회
- 거래 및 포지션 데이터
- 스마트 머니 지갑 추적

<Note>
**예상 시간**: 추천 경로를 완료하는 데 약 1-2시간 소요됩니다
</Note>

## GraphQL 분석가

### 사용 사례

집계 기능이 있는 온체인 OLAP 데이터에 유연하고 선언적인 쿼리가 필요한 개발자 및 분석가.

**대표 시나리오:**
- 여러 체인의 DEX 거래 패턴 분석
- 집계된 온체인 메트릭으로 커스텀 대시보드 구축
- 토큰 홀더 분포 및 지갑 PnL 조회
- 거래, 전송, 풀에 대한 임시 분석 쿼리 수행

**대표 사용자:**
- 온체인 데이터 분석가
- 퀀트 리서처
- 대시보드 및 분석 도구 빌더
- DeFi 프로토콜 팀

### 추천 경로

<Steps>
  <Step title="GraphQL API 이해">
    [GraphQL 개요](/ko/graphql/getting-started/overview)를 읽고 OLAP 분석 API를 이해하세요
  </Step>
  <Step title="첫 번째 쿼리 실행">
    GraphQL IDE를 사용하여 [첫 번째 쿼리 가이드](/ko/graphql/getting-started/first-query)를 따라하세요
  </Step>
  <Step title="데이터 큐브 탐색">
    거래, 전송, 잔액 등을 포괄하는 [12개 데이터 큐브](/ko/graphql/schema/cubes)에 대해 알아보세요
  </Step>
  <Step title="쿼리 예제 시도">
    [DEX 거래 예제](/ko/graphql/examples/dex-trades) 및 기타 쿼리 예제를 둘러보세요
  </Step>
</Steps>

### 핵심 기능

다음 기능을 사용하게 됩니다:
- 모든 온체인 데이터 조회를 위한 단일 GraphQL 엔드포인트
- DEX 거래, 전송, 잔액, 풀, OHLC, 토큰 메트릭을 포괄하는 12개 데이터 큐브
- 내장 집계 (count, sum, avg, min, max, uniq)
- 유연한 필터링, 정렬, 페이지네이션
- 템플릿과 코드 내보내기를 지원하는 브라우저 기반 GraphQL IDE

<Note>
**예상 시간**: 추천 경로를 완료하는 데 약 1-2시간 소요됩니다
</Note>

## 컴플라이언스 사용자

### 사용 사례

거래 리스크 평가 또는 주소 보안 분석이 필요한 팀.

**대표 시나리오:**
- 사용자 입금이 고위험 주소에서 왔는지 탐지
- 지갑 주소의 과거 행동 분석
- 거래 리스크 평가 보고서 생성
- 비즈니스 시스템에 리스크 제어 플로우 통합

**대표 사용자:**
- 거래소 컴플라이언스 팀
- 지갑 보안 팀
- 리스크 제어 시스템 개발자

### 추천 경로

<Steps>
  <Step title="KYT/KYA 이해">
    [보안 컴플라이언스 개요](/ko/guides/security-compliance/overview)를 읽고 리스크 제어 기능을 이해하세요
  </Step>
  <Step title="리스크 평가 API 호출">
    [KYT API 레퍼런스](/ko/api-reference/endpoint/kyt/v1/kyt-transfer-post)를 참고하여 거래 리스크 평가 인터페이스를 이해하세요
  </Step>
  <Step title="실습: 입금 리스크 제어">
    [입금 리스크 제어 통합 튜토리얼](/ko/playbooks/tutorials/integrate-deposit-check)을 따라 전체 플로우를 구현하세요
  </Step>
</Steps>

### 핵심 기능

다음 기능을 사용하게 됩니다:
- KYT 거래 리스크 평가 (리스크 점수 및 라벨 반환)
- KYA 주소 프로파일링 (주소 유형, 과거 행동)
- 리스크 라벨 시스템 (구성 가능한 리스크 규칙)

<Note>
**예상 시간**: 추천 경로를 완료하는 데 약 2-3시간 소요됩니다
</Note>

## AI 빌더

### 사용 사례

AI 에이전트가 온체인 데이터를 조회하고 이해하기를 원하는 개발자.

**대표 시나리오:**
- Claude/GPT가 온체인 데이터 질문에 답할 수 있도록 하기
- 암호화폐 시장을 이해하는 AI 어시스턴트 구축
- AI 워크플로우에 온체인 데이터 쿼리 통합

**대표 사용자:**
- AI 에이전트 개발자
- 자동화 거래 시스템 개발자
- LLM 애플리케이션 개발자
- MCP 통합 개발자

### 추천 경로

<Steps>
  <Step title="AI 기능 이해">
    [AI 인프라 개요](/ko/guides/ai-infrastructure/overview)를 읽고 MCP 프로토콜을 이해하세요
  </Step>
  <Step title="MCP 서버 설정">
    [MCP 설정 가이드](/ko/guides/ai-infrastructure/mcp-server/setup-guide)를 따라 통합을 완료하세요
  </Step>
  <Step title="실습: AI 어시스턴트">
    [AI 트레이딩 어시스턴트 튜토리얼](/ko/playbooks/tutorials/ai-agent-with-mcp)을 따라 완전한 애플리케이션을 구축하세요
  </Step>
</Steps>

### 핵심 기능

다음 기능을 사용하게 됩니다:
- MCP 서버 통합 (Claude Desktop 및 기타 클라이언트 지원)
- AI 친화적 도구 세트 (구조화된 출력)
- 자연어 쿼리 (예: "최근 24시간 ETH 가격 조회")

<Note>
**예상 시간**: 추천 경로를 완료하는 데 약 1-2시간 소요됩니다
</Note>

## AI 에이전트 개발자

### 사용 사례

CLI 또는 Agent Skills를 통해 데이터 접근과 DeFi 실행이 모두 필요한 AI 에이전트를 구축하는 개발자.

**대표 시나리오:**
- 온체인 데이터를 조회하고 거래를 실행하는 AI 에이전트 구축
- 토큰 리서치 및 포트폴리오 관리 자동화
- CI/CD 파이프라인에 ChainStream 통합
- Cursor, Claude Code 또는 Codex에서 Agent Skills 사용

**대표 사용자:**
- AI 에이전트 개발자
- 온체인 워크플로우를 자동화하는 DevOps 엔지니어
- Cursor / Claude Code 파워 유저

### 추천 경로

<Steps>
  <Step title="CLI 설치">
    [CLI 개요](/ko/guides/cli/overview)를 읽고 `npx @chainstream-io/cli login`을 실행하세요
  </Step>
  <Step title="Agent Skills 설치">
    사용 플랫폼에 맞는 [Skills 설치 가이드](/ko/guides/ai-infrastructure/agent-skills/installation)를 따르세요
  </Step>
  <Step title="데이터 쿼리 탐색">
    토큰 리서치 및 지갑 분석을 위한 [chainstream-data skill](/ko/guides/ai-infrastructure/agent-skills/chainstream-data)을 학습하세요
  </Step>
  <Step title="DeFi 실행 (선택사항)">
    스왑 및 토큰 생성을 위한 [chainstream-defi skill](/ko/guides/ai-infrastructure/agent-skills/chainstream-defi)을 학습하세요
  </Step>
</Steps>

### 핵심 기능

다음 기능을 사용하게 됩니다:
- 토큰, 시장, 지갑, DeFi 작업을 위한 CLI 명령어
- 구조화된 AI 워크플로우를 위한 Agent Skills
- 원활한 구독 관리를 위한 x402 자동 결제
- AI 채팅 어시스턴트를 위한 MCP 서버 통합

<Note>
**예상 시간**: 추천 경로를 완료하는 데 약 30-60분 소요됩니다
</Note>
