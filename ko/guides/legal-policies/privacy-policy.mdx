## 개인정보처리방침

<Info>
Chainstream("당사", "우리" 또는 "저희")은 개발자, 분석가, 기관을 위한 실시간 블록체인 데이터 스트리밍 플랫폼을 제공합니다. 이 개인정보처리방침은 당사가 개인정보를 수집, 사용 및 공개하는 방법과 귀하가 개인정보 보호 권리를 행사하는 방법을 이해하는 데 도움을 드리기 위한 것입니다.
</Info>

### 적용 범위

<Note>
이 개인정보처리방침은 당사 웹사이트, 문서 포털, API, 터미널, SDK 및 관련 서비스(총칭하여 "서비스")를 통해 수집하는 개인정보에 적용됩니다.
</Note>

### 개인정보처리방침 변경

<Note>
당사는 언제든지 이 개인정보처리방침을 개정할 수 있습니다. 중요한 변경이 있는 경우 이메일 또는 방침 날짜 업데이트를 통해 알려드립니다. 서비스를 계속 사용하면 업데이트된 조건에 동의하는 것으로 간주됩니다.
</Note>

### 수집하는 정보

<CardGroup cols={1}>
  <Card title="귀하가 제공하는 정보" icon="user">
    - 계정 등록: Chainstream 계정을 등록할 때, 이름, 이메일, 조직명(선택), 비밀번호, 블록체인 지갑 주소(해당되는 경우)를 수집합니다.
    - API 사용: API 키를 요청하거나 유료 기능에 접근할 때, 결제 정보 및 지불 세부 사항을 수집할 수 있습니다.
    - 지원 및 커뮤니케이션: 지원이나 피드백을 위해 연락하시면, 귀하의 커뮤니케이션 내용과 추가로 제공하시는 데이터를 수집합니다.
    - 토큰 제출: 토큰 업데이트를 제출하거나 토큰 정보 업데이트 서비스를 사용하시면, 메타데이터, 토큰 이미지, 소셜 링크 및 관련 정보를 수집할 수 있습니다.
  </Card>

  <Card title="자동으로 수집하는 정보" icon="robot">
    - 로그 데이터: IP 주소, 브라우저 유형, 운영 체제, 참조 URL, 디바이스 식별자, 타임스탬프, API 호출 지표, 사용자 에이전트를 포함합니다.
    - 사용 분석: 성능 향상 및 공정 사용 보장을 위해 API 사용량(토픽 구독, 쿼리 볼륨, 응답 시간 포함)을 자동으로 추적합니다.
    - 쿠키 및 추적 기술: 사용자 경험 향상, 사용 분석 및 콘텐츠 개인화를 위해 쿠키와 유사한 기술을 사용합니다. 여기에는 필수, 분석, 환경설정 쿠키가 포함됩니다.
  </Card>

  <Card title="제3자로부터 수집하는 정보" icon="link">
    - 통합: Chainstream 계정을 제3자 도구(예: 지갑, 거래소)와 통합하면, 해당 서비스에서 허용하는 범위 내에서 관련 데이터를 수신할 수 있습니다.
    - 공개 블록체인 데이터: 지갑 주소 및 트랜잭션 이력을 포함하여 이미 공개적으로 접근 가능한 블록체인 데이터를 인덱싱하고 연관시킵니다.
  </Card>
</CardGroup>

### 정보 사용 방법

<Card title="데이터 사용 목적" icon="chart-mixed">
수집한 정보는 다음 목적으로 사용합니다:
- 서비스의 제공, 유지 관리 및 개선
- 사용자 계정 관리 및 로그인 인증
- API 속도 제한 및 공정 사용 모니터링 및 시행
- 사용자 문의 응대 및 기술 지원 제공
- 서비스 관련 공지 및 커뮤니케이션 발송
- 결제 및 구독 관련 업무 처리
- 내부 분석 수행 및 새로운 기능 개발
- 이용약관 시행 및 부정행위 방지
</Card>

### 귀하의 개인정보 보호 선택 및 권리

  <Card title="계정 정보" icon="user-gear">

  <Card title="커뮤니케이션 환경설정" icon="envelope">

  </Card>

  <Card title="정보 주체의 권리" icon="shield">

    - 당사가 보유한 개인 데이터에 대한 접근
    - 데이터의 정정 또는 삭제 요청
    - 특정 데이터 사용에 대한 이의 제기 또는 제한
    - 데이터 이동성 요청

    이러한 권리를 행사하려면 privacy@chainstream.io로 연락하세요. 귀하의 신원을 확인하고 법률에서 요구하는 대로 응답하겠습니다.
  </Card>

### 중요 고지사항

<CardGroup cols={1}>
  <Card title="데이터 보안" icon="lock">
    <Warning>
    당사는 손실, 오용 및 무단 접근으로부터 데이터를 보호하기 위해 업계 표준의 기술적, 조직적 조치를 사용합니다. 그러나 완전히 안전한 시스템은 없습니다. 귀하는 자격 증명을 기밀로 유지할 책임이 있습니다.
    </Warning>
  </Card>

  <Card title="국제 전송" icon="globe">
    당사의 서비스는 전 세계적으로 운영됩니다. 미국 이외의 지역에서 서비스에 접근하는 경우, 귀하는 데이터 보호법이 다른 미국 또는 기타 국가에서의 데이터 전송, 저장 및 처리에 동의합니다.
  </Card>

  <Card title="데이터 보존" icon="clock">
    당사는 서비스를 제공하고 이 개인정보처리방침에 명시된 목적을 이행하는 데 필요한 기간 동안만 데이터를 보존합니다. 다만, 법률에서 더 긴 보존 기간을 요구하는 경우는 예외입니다.
  </Card>

  <Card title="아동 개인정보" icon="child">
    <Warning>
    Chainstream은 13세 미만의 개인을 대상으로 하지 않으며, 아동으로부터 고의로 개인정보를 수집하지 않습니다. 당사가 해당 데이터를 수집한 것을 인지하게 되면, 즉시 삭제합니다.
    </Warning>
  </Card>
</CardGroup>

</Note>
