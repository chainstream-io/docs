Raydium은 Solana에서 가장 큰 AMM 및 유동성 공급자입니다. ChainStream은 REST API, GraphQL, WebSocket, MCP Tools를 통해 Raydium 거래와 풀의 전체 데이터 커버리지를 제공합니다.

<Info>
**지원 기능**: 거래 분석, 유동성 풀 모니터링, 토큰 가격 추적, OHLCV 데이터, 홀더 분석, Raydium 런치패드를 통한 토큰 생성.
</Info>

## Raydium의 최근 거래를 조회하려면?

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?limit=20" \
  -H "X-API-KEY: your_api_key"
```

전체 파라미터 레퍼런스는 [거래 - 목록](/ko/api-reference/endpoint/data/trade/v2/trade-chain-get)을 참조하세요.
  </Tab>
  <Tab title="GraphQL">

`DEXTrades`를 쿼리하고 Raydium의 프로토콜 이름으로 필터링:

```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    dexProgram: {is: "RAYDIUM_PROGRAM_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
      }
      Dex { ProgramAddress ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", limit: 20 })
```
  </Tab>
</Tabs>

## Raydium에서 토큰 가격을 조회하려면?

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

전체 레퍼런스는 [토큰 - 가격](/ko/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)을 참조하세요.
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

## Raydium에서 토큰을 생성하려면?

ChainStream은 API를 통해 Raydium 런치패드에서 직접 토큰을 생성하는 것을 지원합니다.

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "raydium",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

전체 레퍼런스는 [DEX - 생성](/ko/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)을 참조하세요.
  </Tab>
</Tabs>

<Note>
토큰 생성은 Solana에서만 가능합니다. 프라이빗 키는 생성 트랜잭션 서명에 사용됩니다.
</Note>

## FAQ

<AccordionGroup>
  <Accordion title="Pump.fun 풀과 Raydium 풀의 차이점은?">
    Pump.fun 토큰은 졸업할 때까지 본딩 커브에서 거래됩니다. 졸업 후 유동성은 Raydium AMM 풀로 마이그레이션됩니다. 최근 졸업한 토큰을 찾으려면 `ranking/sol/migrated`를 사용하세요.
  </Accordion>
  <Accordion title="DEX별로 거래를 필터링하려면?">
    GraphQL에서는 `dexProgram` 셀렉터를 사용하여 Raydium의 프로그램 주소로 필터링합니다. REST에서는 거래가 모든 DEX에 걸쳐 반환됩니다 — 응답의 `dex` 필드를 확인하여 Raydium 거래를 식별하세요.
  </Accordion>
  <Accordion title="Raydium 풀 이벤트를 실시간으로 추적할 수 있나요?">
    예. WebSocket 구독을 사용하여 실시간 풀 및 거래 업데이트를 수신할 수 있습니다. 설정 방법은 [실시간 스트리밍](/ko/guides/data-concepts/realtime-streaming)을 참조하세요.
  </Accordion>
</AccordionGroup>
