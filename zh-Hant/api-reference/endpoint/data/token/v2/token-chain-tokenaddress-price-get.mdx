---
title: '代幣 - 按時間查價格'
openapi: '/zh-Hant/api-reference/openapi-data-zh-Hant.yaml GET /v2/token/{chain}/{tokenAddress}/price'
---

<Note>
取得代幣以 USD 與原生貨幣計價的當前價格。如需歷史價格，請使用 `/prices` 端點或 GraphQL `DEXTrades` Cube。
</Note>
