ChainStream CLI 是你透過 SDK、REST、GraphQL 能訪問的那些 [資料產品](/zh-Hant/docs/data-products/overview) 的命令列門面。它按 "和 `jq`、shell 管道、以及開 shell 的 AI Agent 一起用" 的思路設計。

## 安裝

```bash
# macOS / Linux——Homebrew
brew install chainstream-io/tap/chainstream

# npm
npm install -g @chainstream-io/cli

# 直接下二进制
curl -fsSL https://chainstream.io/install.sh | sh
```

## 登入

```bash
chainstream auth login          # 浏览器 OAuth
# 或
export CHAINSTREAM_API_KEY=...  # 脚本里用环境变量
```

## 上手命令

```bash
chainstream tokens search --chain solana --query USDC --limit 5
chainstream trades tail --chain ethereum --token 0xA0b8...
chainstream rankings hot --chain solana --window 24h
chainstream defi quote --chain solana --from SOL --to USDC --amount 1
```

預設輸出 JSON；人類可讀加 `--format=table`。

## 什麼時候用

- 快速探索（"現在發生什麼？"）
- 指令碼、cron、GitHub Actions
- 給 AI Agent 一套可執行的 ChainStream 原語
- 透過管道把 ChainStream 輸出餵給其他 CLI 工具
