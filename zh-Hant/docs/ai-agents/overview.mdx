ChainStream 向 AI Agent 暴露的 [資料產品](/zh-Hant/docs/data-products/overview) 與執行原語和其他客戶端完全一致——只是在鑑權、工具發現與按次付費這三個場景上提供了 Agent 真正需要的形態。

<Note>
  **AI Agents 是消費面，不是獨立產品。** Agent 能做的所有事情都可以透過 REST、GraphQL、
  WebSocket、Kafka、SDK、CLI 獲取。本分割槽文件講的是讓 LLM 能夠順暢消費能力目錄的那層外形。
</Note>

## 兩大主要接入面

<CardGroup cols={2}>
  <Card title="MCP Server" icon="robot" href="/zh-Hant/docs/ai-agents/mcp-server/introduction">
    託管（也可自託管）的 Model Context Protocol 服務，把 ChainStream 能力目錄暴露為型別化工具。
    Claude、Cursor、ChatGPT、Windsurf 等 MCP 相容客戶端開箱即用。
  </Card>
  <Card title="Agent Skills" icon="brain" href="/zh-Hant/docs/ai-agents/agent-skills/introduction">
    可安裝的 SKILL.md 能力包，教 Agent <em>怎麼</em> 用 ChainStream——代幣分析、
    Smart Money 跟蹤、DeFi 執行等模式，面向 Cursor、Claude Code、Codex。
  </Card>
</CardGroup>

## 為什麼這件事對 Agent 重要

傳統 LLM 接入區塊鏈有三個老問題：模型有知識截止日期、沒有鑑權過的資料來源、無法在未預註冊賬戶的情況下付費讀取。ChainStream 逐個解決：

<AccordionGroup>
  <Accordion title="實時資料，天生帶 schema 好給 tool call 用" icon="bolt">
    每個資料產品都能以結構化 tool call 接入，返回 JSON schema。
    Agent 拿到的是規範化 ID，而不是被截圖出來的文字——後續呼叫可以直接串起來，
    不必每一步都重解析。
  </Accordion>
  <Accordion title="鑑權貼合 Agent 的執行迴圈" icon="shield-halved">
    API Key 可用、OAuth 可用；自治 Agent 可用
    [x402 / MPP](/zh-Hant/docs/platform/billing-payments/x402-payments)——這是一種按次付費的 HTTP 支付協議，
    Agent 用 USDC 付款、立即呼叫，無需事先開戶。
  </Accordion>
  <Accordion title="可選的寫路徑帶使用者級保護" icon="right-left">
    DEX 報價、路由、swap 提交都可用。寫操作必須由使用者控制的錢包簽名——
    Agent 可以構造交易，但不會代表別人的錢默默傳送。
  </Accordion>
</AccordionGroup>

## 我該選哪個？

| 情況 | 接入面 |
|:--|:--|
| 你在用 Claude / Cursor / ChatGPT，想接入 ChainStream 工具 | [MCP Server](/zh-Hant/docs/ai-agents/mcp-server/introduction) |
| 你想裝一個現成的 Agent Skill 做代幣研究 | [Agent Skills](/zh-Hant/docs/ai-agents/agent-skills/chainstream-data) |
| 你自己用 Python / TS 寫 Agent，想直接調 | [SDK](/zh-Hant/sdks/overview) |
| 你想按次用 USDC 付費，不註冊賬號 | [x402 支付](/zh-Hant/docs/platform/billing-payments/x402-payments) |
| 你想用 Tempo / Stripe / 穩定幣付費 | [MPP 支付](/zh-Hant/docs/platform/billing-payments/mpp-payments) |
| 你想給 Agent 一套 shell 原語 | [CLI](/zh-Hant/docs/access-methods/cli) |

## 面向 Agent 的 Quickstart

<Steps>
  <Step title="啟動 MCP Server">
    ```bash
    CHAINSTREAM_API_KEY=your_api_key npx @chainstream-io/mcp
    ```
  </Step>
  <Step title="把你的 Agent 指過來">
    Claude Desktop、Cursor、ChatGPT 的完整配置片段見
    [MCP Server 安裝](/zh-Hant/docs/ai-agents/mcp-server/setup)。
  </Step>
  <Step title="提問">
    *"列出 24 小時內 Solana 交易量 Top 10 的代幣，並標出有 KYT 告警的那些。"*
    Agent 會自己選工具、串起呼叫。
  </Step>
</Steps>
