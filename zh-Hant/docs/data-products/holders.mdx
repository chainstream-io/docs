**Holders** 資料產品對外暴露每個代幣的持幣人表與分佈指標——你可以回答"誰持有這個代幣"、"集中度有多高"、"誰剛剛動了"，不用去抓瀏覽器。

## 包含什麼

| 欄位組 | 舉例 |
|:--|:--|
| 持幣表 | `rank`, `address`, `balance`, `shareOfSupply`, `entityLabel`, `costBasisUsd` |
| 分佈 | `giniCoefficient`, `top10Share`, `top100Share` |
| 變化 | `changesLast24h`（新增、退出、排名變化） |
| 分類 | `bucket`——大戶 / 散戶 / 合約 / 交易所 / 流動性池 |

## 覆蓋與新鮮度

- 覆蓋 [支援的鏈](/zh-Hant/docs/supported-chains) 全部。
- 持幣餘額在命中轉賬事件之後秒級重新整理。
- 分佈指標每 5 分鐘重算。

## 常見用例

- 檢測大戶吸籌 / 派發
- 空投資格快照
- 代幣盡調的集中度風險評分
- Top-10 佔比隨時間的圖表

## 計費

每次呼叫 **1 read unit**；大持幣表分頁返回。
