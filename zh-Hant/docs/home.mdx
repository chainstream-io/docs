## ChainStream 能給你什麼

### 統一的多鏈資料

覆蓋全部支援鏈的單一 schema。代幣價格、DEX 交易、錢包 PnL、池子流動性、持幣人分佈、OHLC K 線、Smart Money 活動、區塊鏈底層資料——全部透過同一套鑑權、同一種 ID、同一組維度過濾器訪問。

### 匹配你技術棧的實時投遞

<CardGroup cols={3}>
  <Card title="REST + GraphQL" icon="bolt">
    請求-響應模式。適合儀表盤、使用者側查詢與臨時分析。
  </Card>
  <Card title="WebSocket" icon="tower-broadcast">
    亞秒級推送到瀏覽器、App 與需要實時 UI 更新的後端。
  </Card>
  <Card title="Kafka Streams" icon="bars-staggered">
    對交易引擎、索引器、告警流水線等延遲敏感後端的 exactly-once 投遞。
  </Card>
</CardGroup>

### 內建合規

KYT（Know Your Transaction）與 KYA（Know Your Address）就在同一套 API 內。給一筆轉賬打分、給一個對手方畫像、生成 Travel Rule 格式報告——不用第二個供應商、不用第二份賬單、不多一個資料外洩面。

### 對 AI Agent 一等公民支援

AI Agent 與其他客戶端使用相同介面接入 ChainStream。託管的 MCP Server 將我們的能力目錄暴露為型別化工具；Agent Skills 以可安裝的 SKILL.md 包形式釋出；對 Agent 友好的鑑權（API Key、OAuth、x402/MPP 微支付）讓 Agent 可以按次付費呼叫，無需預先開戶。
