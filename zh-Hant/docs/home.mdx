## 什麼是 ChainStream

ChainStream 是為 Web3 時代構建的 AI-Native 基礎設施。與傳統 Web3 資料 API 不同，ChainStream 從第一天起就為 AI Agent 和自動化場景設計，提供：

- **統一資料層**: 一個 API 獲取多鏈資料，無需對接多個供應商
- **內建合規能力**: KYT/KYA 原生整合，無需第三方風控服務
- **AI 執行層**: 透過 MCP 協議讓 AI Agent 直接操作鏈上世界

<Tip>
讓開發者和 AI Agent 能夠安全、高效地訪問和操作鏈上世界。
</Tip>

## 核心能力

ChainStream 提供三大核心能力，覆蓋 Web3 資料獲取、安全合規和 AI 執行的完整鏈路：

### Data — 實時鏈上資料

透過統一 API 訪問多鏈實時資料，支援 WebSocket 實時推送和歷史資料查詢。

<AccordionGroup>
  <Accordion title="多鏈資料聚合">
    一個 API 端點獲取 Solana、Ethereum、BSC 資料，無需分別對接多個資料來源。支援跨鏈查詢和資料標準化。
  </Accordion>
  <Accordion title="實時價格與交易資料">
    毫秒級價格更新，支援 DEX/CEX 聚合價格、交易對深度、K線資料。提供 WebSocket 實時推送，延遲 &lt;2ms。
  </Accordion>
  <Accordion title="Smart Money 錢包追蹤">
    追蹤鏈上聰明錢動向，包括鯨魚錢包、機構地址、KOL 錢包。提供錢包標籤、持倉變化、交易行為分析。
  </Accordion>
  <Accordion title="歷史資料查詢">
    支援任意時間範圍的歷史資料查詢，包括歷史價格、歷史持倉、歷史交易記錄。資料保留期限根據套餐不同。
  </Accordion>
</AccordionGroup>

### Security — 安全與合規

內建企業級安全合規能力，滿足交易所、錢包、DeFi 協議的風控需求。

<AccordionGroup>
  <Accordion title="KYT (Know Your Transaction) 交易風險評估">
    實時分析交易風險等級，識別可疑資金流向。支援 OFAC 制裁名單檢查、混幣器識別、暗閘道器聯檢測。返回風險評分 (0-100) 和詳細風險標籤。
  </Accordion>
  <Accordion title="KYA (Know Your Address) 地址畫像分析">
    全面的地址畫像分析，包括地址型別識別（EOA/合約/多籤）、歷史行為分析、資金來源追溯、關聯地址圖譜。
  </Accordion>
  <Accordion title="合規審計支援">
    提供完整的審計日誌，滿足監管報告需求。支援匯出 FATF Travel Rule 格式報告，整合主流合規工具（Chainalysis、Elliptic 格式相容）。
  </Accordion>
</AccordionGroup>

### AI Execution — AI 執行層

透過 MCP (Model Context Protocol) 讓 AI Agent 安全地與區塊鏈互動。

<AccordionGroup>
  <Accordion title="MCP Server 支援">
    原生支援 Anthropic MCP 協議，讓 Claude 等 AI 模型直接呼叫 ChainStream 能力。無需編寫適配程式碼，開箱即用。
  </Accordion>
  <Accordion title="AI Agent 友好的工具集">
    專為 AI Agent 設計的工具介面，包括結構化輸出、錯誤自愈、上下文保持。支援 LangChain、AutoGPT、CrewAI 等主流框架。
  </Accordion>
  <Accordion title="自然語言到鏈上操作">
    AI 可透過自然語言指令完成鏈上查詢和操作，如"查詢 0x... 地址最近 7 天的交易"、"監控 PEPE 價格超過 $0.00001 時提醒我"。
  </Accordion>
</AccordionGroup>

<Warning>
**安全提示**: AI 執行層僅支援讀取操作和預授權的寫入操作，所有敏感操作需要使用者確認。
</Warning>
