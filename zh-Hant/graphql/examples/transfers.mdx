**Transfers** Cube 收錄鏈上代幣轉帳事件 — 每次代幣從一個錢包移到另一個錢包。可用來追蹤錢包活動、監控大戶動向、分析代幣流向，以及偵測交易所存提款。

<Note>
以下範例皆使用 `network: sol`（Solana）。若為其他支援鏈，請改為 `eth`、`bsc` 或 `polygon`。
</Note>

---

## 如何取得最新的轉帳？

取得 Solana 上最近 10 筆代幣轉帳，包含發送方、接收方、數量與美元價值。

```graphql
query {
  Solana {
    Transfers(
      limit: {count: 10}
      orderBy: {descending: Block_Time}
    ) {
      Block { Time }
      Transaction { Hash }
      Transfer {
        Currency { MintAddress }
        Sender { Address }
        Receiver { Address }
        Amount
        AmountInUSD
      }
    }
  }
}
```

<Tip>
[在 GraphQL IDE 開啟](https://ide.chainstream.io) — 貼上上方查詢即可互動執行，並享有自動完成與 schema 探索。
</Tip>

<AccordionGroup>
  <Accordion title="主要欄位">
    | 欄位 | 說明 |
    |:------|:------------|
    | `Block.Time` | 區塊時間戳（ISO 8601） |
    | `Transaction.Hash` | 鏈上交易雜湊 |
    | `Transfer.Currency.MintAddress` | 被轉移的代幣地址 |
    | `Transfer.Sender.Address` | 發送代幣的錢包 |
    | `Transfer.Receiver.Address` | 接收代幣的錢包 |
    | `Transfer.Amount` | 轉移的代幣數量 |
    | `Transfer.AmountInUSD` | 發生當下的轉帳美元價值 |
  </Accordion>
  <Accordion title="自訂建議">
    - **依代幣篩選**：加上 `tokenAddress: {is: "TOKEN_ADDRESS"}` 僅看特定代幣
    - **僅大額轉帳**：加上 `where: {Transfer: {AmountInUSD: {gt: 10000}}}` 找出大戶動向
    - **時間範圍**：加上 `where: {Block: {Time: {after: "2025-03-01T00:00:00Z"}}}` 限定區間
  </Accordion>
</AccordionGroup>

## 如何取得某錢包對外轉出的轉帳？

使用 `senderAddress` 選擇器取得特定錢包對外發出的轉帳。

```graphql
query {
  Solana {
    Transfers(
      limit: {count: 20}
      senderAddress: {is: "WALLET_ADDRESS"}
      orderBy: {descending: Block_Time}
    ) {
      Block { Time }
      Transfer {
        Currency { MintAddress }
        Receiver { Address }
        Amount
        AmountInUSD
      }
    }
  }
}
```

<Tip>
請將 `WALLET_ADDRESS` 換成要調查的實際錢包地址。
</Tip>

<AccordionGroup>
  <Accordion title="主要欄位">
    | 欄位 | 說明 |
    |:------|:------------|
    | `Transfer.Currency.MintAddress` | 送出的代幣 |
    | `Transfer.Receiver.Address` | 接收方 |
    | `Transfer.Amount` | 送出的代幣數量 |
    | `Transfer.AmountInUSD` | 轉帳當下的美元價值 |
  </Accordion>
  <Accordion title="自訂建議">
    - **僅單一代幣**：加上 `tokenAddress: {is: "TOKEN_ADDRESS"}` 縮小範圍
    - **增加筆數**：將 `count: 20` 改為最多 `10000` 以拉更深歷史
    - **加入接收方篩選**：使用 `where: {Transfer: {Receiver: {Address: {is: "RECEIVER_ADDRESS"}}}}` 追蹤兩個特定錢包之間的轉帳
  </Accordion>
</AccordionGroup>

## 如何取得轉入某錢包的轉帳？

使用 `receiverAddress` 選擇器找出轉入該錢包的轉帳。

```graphql
query {
  Solana {
    Transfers(
      limit: {count: 20}
      receiverAddress: {is: "WALLET_ADDRESS"}
      orderBy: {descending: Block_Time}
    ) {
      Block { Time }
      Transfer {
        Currency { MintAddress }
        Sender { Address }
        Amount
        AmountInUSD
      }
    }
  }
}
```

<Tip>
Transfers Cube 同時支援 `senderAddress` 與 `receiverAddress` 選擇器。可與 `where` 併用做額外篩選（例如時間範圍或最小金額）。
</Tip>

## 如何取得特定代幣的所有轉帳？

追蹤網路上某代幣的所有轉帳。

```graphql
query {
  Solana {
    Transfers(
      limit: {count: 20}
      tokenAddress: {is: "TOKEN_ADDRESS"}
      orderBy: {descending: Block_Time}
    ) {
      Block { Time }
      Transaction { Hash }
      Transfer {
        Sender { Address }
        Receiver { Address }
        Amount
        AmountInUSD
      }
    }
  }
}
```

<AccordionGroup>
  <Accordion title="自訂建議">
    - **大戶警示**：加上 `where: {Transfer: {AmountInUSD: {gt: 100000}}}` 僅看大額轉帳（>10 萬美元）
    - **排除碎額**：加上 `where: {Transfer: {Amount: {gt: 0.01}}}` 過濾可忽略的小額
    - **限定時間**：可與 `where: {Block: {Time: {since: "2025-03-27T00:00:00Z"}}}` 併用取得當日轉帳
  </Accordion>
</AccordionGroup>
