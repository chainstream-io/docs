## 前置需求

開始前請確認：

- 已註冊 ChainStream 帳號（[在此註冊](https://www.chainstream.io/dashboard)）
- 已取得 API Key（格式為 `cs_live_...`）

<Tip>
GraphQL API 與 REST Data API 共用同一組 API Key。若你已有 REST API 的 Key，可直接沿用。
</Tip>

---

## 步驟一：取得 API Key

1. 登入 [ChainStream Dashboard](https://www.chainstream.io/dashboard)
2. 前往 **Applications**
3. 點選 **Create New App**（或選擇既有應用）
4. 複製你的 **API Key**

---

## 步驟二：開啟 GraphQL IDE

請開啟 ChainStream GraphQL IDE：

<Card title="GraphQL IDE" icon="code" href="https://ide.chainstream.io">
  https://ide.chainstream.io
</Card>

IDE 提供自動完成、語法突顯、內嵌文件與查詢歷程——是探索 schema 與撰寫查詢最快的方式。

---

## 步驟三：設定驗證

在 IDE 中開啟 **Headers** 面板（左下角），並加入 API Key：

```json
{
  "X-API-KEY": "your_api_key"
}
```

---

## 步驟四：執行範例查詢

將下列查詢貼入編輯器。此查詢會取得 **Solana 上最新 10 筆 DEX 成交**，含區塊時間、交易雜湊、買／賣代幣資訊、數量、USD 價格與 DEX 協定名稱。

```graphql
query {
  Solana {
    DEXTrades(
      limit: {count: 10}
      orderBy: {descending: Block_Time}
    ) {
      Block {
        Time
        Slot
      }
      Transaction {
        Hash
      }
      Trade {
        Buy {
          Currency { MintAddress }
          Amount
          PriceInUSD
        }
        Sell {
          Currency { MintAddress }
          Amount
        }
        Dex { ProtocolName }
      }
    }
  }
}
```

<Tip>
[在 GraphQL IDE 中開啟](https://ide.chainstream.io)——貼上上方查詢即可互動執行，並享有自動完成與 schema 探索。
</Tip>

點選 **Play** 按鈕（或按下 `Ctrl+Enter`／`Cmd+Enter`）執行。

---

## 等同的 cURL 指令

你也可以在終端機執行相同查詢：

```bash
curl -X POST "https://graphql.chainstream.io/graphql" \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: your_api_key" \
  -d '{
    "query": "{ Solana { DEXTrades(limit: {count: 10}, orderBy: {descending: Block_Time}) { Block { Time Slot } Transaction { Hash } Trade { Buy { Currency { MintAddress } Amount PriceInUSD } Sell { Currency { MintAddress } Amount } Dex { ProtocolName } } } } }"
  }'
```

## 理解回應內容

回應結構會對應你的查詢：

| 路徑 | 說明 |
|:--|:--|
| `Block.Time` | 區塊時間戳（ISO 8601） |
| `Block.Slot` | Solana slot 編號（Solana 專用） |
| `Transaction.Hash` | 鏈上交易雜湊 |
| `Trade.Buy.Currency.MintAddress` | 買入資產的代幣位址 |
| `Trade.Buy.Amount` | 買入代幣數量 |
| `Trade.Buy.PriceInUSD` | 成交當下買入代幣的 USD 價格 |
| `Trade.Sell.Currency.MintAddress` | 賣出資產的代幣位址 |
| `Trade.Sell.Amount` | 賣出代幣數量 |
| `Trade.Dex.ProtocolName` | 執行成交的 DEX 協定（例如 Raydium、Orca、PancakeSwap） |

<Info>
`extensions.credits` 物件會顯示此查詢消耗的計費點數與剩餘額度。詳見 [計費與點數](/zh-Hant/graphql/billing/graphql-billing)。
</Info>

## 試著修改查詢

查詢可運作後，可嘗試下列變化：

<AccordionGroup>
  <Accordion title="改查 Ethereum">
    將 `network: sol` 改為 `network: eth` 以查詢 Ethereum DEX 成交（Uniswap、SushiSwap 等）。
  </Accordion>
  <Accordion title="加入時間篩選">
    加入 `where` 子句，篩選過去一小時內的成交：
    ```graphql
    Solana {
      DEXTrades(
        limit: {count: 10}
        orderBy: {descending: Block_Time}
        where: {Block: {Time: {after: "2025-03-27T09:00:00Z"}}}
      ) { ... }
    }
    ```
  </Accordion>
  <Accordion title="查詢特定代幣">
    依代幣 mint 位址篩選，只看該代幣的成交：
    ```graphql
    Solana {
      DEXTrades(
        limit: {count: 10}
        orderBy: {descending: Block_Time}
        where: {Trade: {Buy: {Currency: {MintAddress: {is: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"}}}}}
      ) { ... }
    }
    ```
  </Accordion>
  <Accordion title="加入聚合">
    統計各 DEX 協定的總成交筆數：
    ```graphql
    Solana {
      DEXTrades(
        where: {Block: {Time: {after: "2025-03-27T00:00:00Z"}}}
      ) {
        count
        Trade { Dex { ProtocolName } }
      }
    }
    ```
  </Accordion>
</AccordionGroup>
