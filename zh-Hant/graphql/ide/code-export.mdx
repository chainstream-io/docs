## 概覽

「程式碼匯出」功能會將你目前的 GraphQL 查詢轉成所選語言的可直接使用的程式碼片段。產生的程式碼包含完整 HTTP 請求設定——端點 URL、驗證標頭、查詢本文與變數——因此可直接複製到專案中使用。

---

| 語言 | 格式 | 使用情境 |
|:--|:--|:--|
| **cURL** | Shell 指令 | 快速終端機測試、CI/CD 腳本 |

| **PHP** | cURL 擴充 | PHP 後端、WordPress 外掛 |

## 如何匯出

<Steps>
  <Step title="撰寫查詢">
    在編輯器中輸入或載入查詢。匯出前請先確認查詢可成功執行。
  </Step>
  <Step title="點選 Export Code">
    點選工具列上的 **Export Code** 按鈕。會開啟對話框，內含語言選擇器與產生的片段。
  </Step>
  <Step title="選擇語言">
    從下拉選單選擇目標語言。片段會立即更新。
  </Step>
  <Step title="複製片段">
    點選 **Copy** 按鈕，將產生的程式碼複製到剪貼簿。
  </Step>
</Steps>

<Tip>
匯出的程式碼會使用你目前在 IDE 中設定的 GraphQL 端點與 API Key。若已自訂端點或標頭，產生的片段會一併反映。
</Tip>

## 範例片段

下列範例匯出簡單的 DEXTrades 查詢——取得最新 10 筆 Solana DEX 成交：

```graphql
query {
  Solana {
    DEXTrades(
      limit: {count: 10}
      orderBy: {descending: Block_Time}
    ) {
      Block { Time }
      Transaction { Hash }
      Trade {
        Buy { Currency { MintAddress } Amount PriceInUSD }
        Sell { Currency { MintAddress } Amount }
        Dex { ProtocolName }
      }
    }
  }
}
```

    import (

    use reqwest::header::{HeaderMap, HeaderValue, CONTENT_TYPE};
    use serde_json::json;

    #[tokio::main]
    async fn main() -> Result<(), reqwest::Error> {
        let mut headers = HeaderMap::new();
        headers.insert(CONTENT_TYPE, HeaderValue::from_static("application/json"));
        headers.insert("X-API-KEY", HeaderValue::from_static("your_api_key"));

        let body = json!({
            "query": r#"{ Solana { DEXTrades(limit: {count: 10}, orderBy: {descending: Block_Time}) { Block { Time } Transaction { Hash } Trade { Buy { Currency { MintAddress } Amount PriceInUSD } Sell { Currency { MintAddress } Amount } Dex { ProtocolName } } } } }"#

        let client = reqwest::Client::new();
        let res = client
            .post("https://graphql.chainstream.io/graphql")
            .headers(headers)
            .json(&body)
            .send()
            .await?;

        println!("{}", res.text().await?);
        Ok(())
    }

## 匯出程式碼包含哪些內容

每個產生的片段都包含：

| 項目 | 說明 |
|:--|:--|
| **Endpoint URL** | `https://graphql.chainstream.io/graphql` |
| **驗證** | 來自 IDE Headers 面板的 `X-API-KEY` 標頭 |
| **Content-Type** | `application/json` |
| **查詢本文** | 編輯器中的完整 GraphQL 查詢 |
| **變數** | 若 Variables 面板有值則一併包含 |

<Note>
執行匯出程式碼前，請將 `your_api_key` 換成實際的 API Key。基於安全，IDE 不會在匯出片段中嵌入真實金鑰。
</Note>

## 建議

- **先測試再匯出**——先在 IDE 執行查詢確認可用。匯出程式碼會原樣複製查詢內容。
- **以變數參數化**——使用 GraphQL 變數取代硬編碼值。匯出程式碼會同時包含 `variables` JSON 與查詢。
- **檢查相依套件**——Python 片段使用 `requests`、Rust 使用 `reqwest`、Go 使用標準函式庫。請確認專案已安裝所需套件。
