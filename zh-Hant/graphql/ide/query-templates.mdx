## 什麼是查詢模板

查詢模板是預寫好的 GraphQL 查詢，覆蓋 Solana、Ethereum 與 BSC 上最常見的分析場景。每個模板針對特定 Cube，已帶好正確的查詢結構、欄位與篩選引數 — 替換佔位值後即可執行。

模板幫助你：

- **透過示例學習 schema** — 瞭解欄位、篩選與排序在實際中如何配合
- **快速上手** — 無需從零編寫查詢
- **發現** 你可能尚未了解的 **Cube**

---

## 如何訪問模板

<Steps>
  <Step title="開啟側邊欄">
    點選側邊欄開關或按 **Ctrl/Cmd+B** 開啟左側面板。
  </Step>
  <Step title="選擇 Queries 標籤">
    側邊欄有兩個標籤：**Queries**（模板）與 **Saved**（已儲存查詢）。選擇 **Queries**。
  </Step>
  <Step title="按鏈與 Cube 瀏覽">
    模板以樹形組織：**鏈**（Solana / Ethereum / BSC）→ **Cube**（DEXTrades、Transfers 等）→ **模板**。
  </Step>
  <Step title="點選載入">
    點選任意模板即可載入查詢編輯器。佔位值會高亮，便於替換。
  </Step>
</Steps>

---

## 模板組織方式

模板採用兩級層級：

```
Chain (Solana / Ethereum / BSC)
 └── Cube
      └── Template
```

每條鏈包含同一組 Cube 與模板，並已配置正確的 `network` 值（`sol`、`eth` 或 `bsc`）。

## 佔位符

模板用大寫佔位符標出執行前需替換為真實鏈上地址的值。

| 佔位符 | 說明 | 示例 |
|:--|:--|:--|
| `TOKEN_ADDRESS` | 代幣合約 / mint 地址 | `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`（Solana 上 USDC） |
| `WALLET_ADDRESS` | 錢包公鑰地址 | `5Q544fKrFoe6tsEbD7S8EmxGTJYAKtTVhAW5Q5pge4j1` |
| `POOL_ADDRESS` | DEX 流動性池地址 | `58oQChx4yWmvKdwLLZzBi4ChoCc2fqCUWBkwMihLYQo2` |

<Note>
佔位符在查詢中以字面字串出現（例如 `MintAddress: {is: "TOKEN_ADDRESS"}`）。請將整段佔位字串（含引號）替換為你的真實地址。
</Note>

## 示例：使用模板

以下演示如何在 Solana 上使用 **DEXTrades → Latest Trades** 模板：

<Steps>
  <Step title="開啟側邊欄並選擇模板">
    進入 **Solana → DEXTrades → Latest Trades** 並點選載入。
  </Step>
  <Step title="檢查載入的查詢">
    編輯器會填入：
    ```graphql
    query {
      DEXTrades(
        network: sol
        limit: {count: 10}
        orderBy: Block_Time_DESC
      ) {
        Block { Time Slot }
        Transaction { Hash }
        Trade {
          Buy {
            Currency { MintAddress Symbol Name }
            Amount
            PriceInUSD
          }
          Sell {
            Currency { MintAddress Symbol Name }
            Amount
          }
          Dex { ProtocolName }
        }
      }
    }
    ```
  </Step>
  <Step title="執行">
    該模板無佔位符 — 會拉取最近 10 筆成交。按 **Ctrl/Cmd+Enter** 執行。
  </Step>
  <Step title="迭代">
    可修改查詢以增加篩選、調整 limit 或選擇不同欄位。schema 自動補全有助於發現可用選項。
  </Step>
</Steps>
