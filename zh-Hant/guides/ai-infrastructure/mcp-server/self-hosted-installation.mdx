## 包

ChainStream MCP Server 以 npm 包形式分發：

```
@chainstream-io/mcp
```

## 安裝

<Tabs>
  <Tab title="全域性安裝">
    全域性安裝以使 `chainstream-mcp` 二進位制在系統範圍內可用：

    ```bash
    npm install -g @chainstream-io/mcp
    ```

    安裝後直接執行：

    ```bash
    chainstream-mcp
    ```
  </Tab>
  <Tab title="npx（免安裝）">
    無需永久安裝，按需執行 MCP Server：

    ```bash
    npx @chainstream-io/mcp
    ```

    每次下載並執行最新版本。適合快速測試。
  </Tab>
</Tabs>

## 二進位制

該包暴露一個二進位制：

```
chainstream-mcp
```

該二進位制預設使用 stdio 傳輸啟動 MCP Server — 這是連線 Claude Desktop、Cursor 和其他 MCP 客戶端的標準模式。詳細配置請參閱 [Stdio 傳輸](/zh-Hant/guides/ai-infrastructure/mcp-server/stdio-transport)。

## 環境變數

| 變數 | 必填 | 預設值 | 說明 |
|------|------|--------|------|
| `CHAINSTREAM_API_KEY` | 是 | — | 你的 ChainStream API Key |
| `CHAINSTREAM_API_URL` | 否 | `https://api.chainstream.io` | ChainStream API 基礎 URL |

將 API Key 設為環境變數或透過命令列傳入：

```bash
export CHAINSTREAM_API_KEY=your-key
chainstream-mcp

# 或直接传入
chainstream-mcp --api-key your-key
```

<Warning>
  請妥善保管你的 API Key。切勿將其提交到版本控制。
</Warning>

## 驗證安裝

安裝後驗證包是否可用：

```bash
npx @chainstream-io/mcp --help
```

你應該看到可用選項：

```
Options:
  --transport  Transport type: stdio for local CLI   [choices: "stdio"] [default: "stdio"]
  --api-key    ChainStream API key                   [string]
  --help       Show help                             [boolean]
  --version    Show version number                   [boolean]
```
