## 隱私政策

<Info>
Chainstream（"我們"、"我們的"或"我們公司"）提供實時區塊鏈資料流平臺，服務物件包括開發者、分析師和機構。本隱私政策旨在幫助您瞭解我們如何收集、使用和披露您的個人資訊，以及您如何行使隱私權。
</Info>

### 適用範圍

<Note>
本隱私政策適用於我們透過網站、文件門戶、API、終端、SDK 和任何相關服務（統稱為"服務"）收集的個人資訊。
</Note>

### 政策變更

<Note>
我們可能隨時修改本隱私政策。如有重大變更，我們將透過電子郵件通知您或更新政策日期。您繼續使用我們的服務即表示您接受更新後的條款。
</Note>

### 資訊收集

<CardGroup cols={1}>
  <Card title="您提供的資訊" icon="user">
    - 賬戶註冊：當您註冊 Chainstream 賬戶時，我們會收集您的姓名、電子郵件、組織名稱（可選）、密碼和區塊鏈錢包地址（如適用）。
    - API 使用：當您請求 API 金鑰或訪問付費功能時，我們可能會收集賬單資訊和支付詳情。
    - 支援和通訊：如果您聯絡我們尋求支援或提供反饋，我們會收集您的通訊內容和您提供的任何其他資料。
    - 代幣提交：如果您提交代幣更新或使用我們的代幣資訊更新服務，我們可能會收集後設資料、代幣圖片、社交連結和相關資訊。
  </Card>

  <Card title="自動收集的資訊" icon="robot">
    - 日誌資料：包括 IP 地址、瀏覽器型別、作業系統、引用 URL、裝置識別符號、時間戳、API 呼叫指標和使用者代理。
    - 使用分析：我們自動跟蹤 API 使用情況（包括主題訂閱、查詢量和響應時間）以改進效能並確保公平使用。
    - Cookie 和跟蹤技術：我們使用 cookie 和類似技術來增強使用者體驗、分析使用情況和個性化內容。這包括必要的、分析性的和偏好性的 cookie。
  </Card>

  <Card title="第三方資訊" icon="link">
    - 整合：當您將 Chainstream 賬戶與第三方工具（如錢包、交易所）整合時，我們可能會根據這些服務的許可接收相關資料。
    - 公開可用的區塊鏈資料：我們索引和關聯已經公開可訪問的區塊鏈資料，包括錢包地址和交易歷史。
  </Card>
</CardGroup>

### 資訊使用

<Card title="資料使用目的" icon="chart-mixed">
我們使用收集的資訊來：
- 提供、維護和改進我們的服務
- 管理使用者賬戶和驗證登入
- 監控和執行 API 速率限制和公平使用
- 響應使用者查詢並提供技術支援
- 傳送服務相關通知和通訊
- 處理賬單和訂閱相關任務
- 進行內部分析和開發新功能
- 執行我們的服務條款並防止欺詐
</Card>

    - 請求資料可攜帶性
