Pump.fun 是 Solana 上領先的 memecoin 發射臺。ChainStream 透過 REST API、GraphQL、WebSocket 與 MCP 工具，為 Pump.fun 代幣提供完整資料覆蓋。

<Info>
**支援的能力**：代幣發現、價格跟蹤、成交分析、持倉者分析、透過 `pumpfun_create` 建立代幣，以及畢業/遷移跟蹤。
</Info>

---

## 如何發現新上線的 Pump.fun 代幣？

使用排行榜介面查詢 Pump.fun 發射臺上新建立的代幣。

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/ranking/sol/newTokens" \
  -H "X-API-KEY: your_api_key"
```

完整引數說明見 [Ranking - New Tokens](/zh-Hant/api-reference/endpoint/data/ranking/v2/ranking-chain-newtokens-get)。
  </Tab>
  <Tab title="GraphQL">

在 Solana 上查詢 `TokenSupplyUpdates`，並篩選 Pump.fun 代幣：

```graphql
query {
  TokenSupplyUpdates(
    network: sol
    limit: {count: 20}
    orderBy: Block_Time_DESC
  ) {
    Block { Time }
    TokenSupplyUpdate {
      Currency {
        MintAddress
        Decimals
      }
      PostBalance
      MarketCapInUSD
      PriceInUSD
      TotalSupply
    }
  }
}
```

<Tip>
在 [GraphQL IDE](https://ide.chainstream.io) 中開啟即可互動式執行上述查詢。
</Tip>
  </Tab>
  <Tab title="MCP Tool">
```
ranking_get_new_tokens({ chain: "solana" })
```

完整工具說明見 [MCP 工具目錄](/zh-Hant/guides/ai-infrastructure/mcp-server/tools-catalog)。
  </Tab>
</Tabs>

## 如何獲取 Pump.fun 代幣價格？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/token/sol/TOKEN_ADDRESS/price" \
  -H "X-API-KEY: your_api_key"
```

完整說明見 [Token - Price](/zh-Hant/api-reference/endpoint/data/token/v2/token-chain-tokenaddress-price-get)。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 1}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    where: {IsSuspect: {eq: false}}
    orderBy: Block_Time_DESC
  ) {
    Trade {
      Buy { PriceInUSD PriceInNative }
    }
    Block { Time }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
token_get_price({ chain: "solana", tokenAddress: "TOKEN_ADDRESS" })
```
  </Tab>
</Tabs>

<Note>
將 `TOKEN_ADDRESS` 替換為實際的 Pump.fun 代幣 mint 地址。
</Note>

## 如何獲取 Pump.fun 代幣的成交記錄？

<Tabs>
  <Tab title="REST API">
```bash
curl "https://api.chainstream.io/v2/trade/sol?tokenAddress=TOKEN_ADDRESS&limit=20" \
  -H "X-API-KEY: your_api_key"
```

完整說明見 [Trade - List](/zh-Hant/api-reference/endpoint/data/trade/v2/trade-chain-get)。
  </Tab>
  <Tab title="GraphQL">
```graphql
query {
  DEXTrades(
    network: sol
    limit: {count: 20}
    tokenAddress: {is: "TOKEN_ADDRESS"}
    orderBy: Block_Time_DESC
  ) {
    Block { Time Slot }
    Transaction { Hash }
    Trade {
      Buy {
        Currency { MintAddress }
        Amount
        PriceInUSD
        Account { Owner }
      }
      Sell {
        Currency { MintAddress }
        Amount
        Account { Owner }
      }
      Dex { ProtocolName }
    }
    Pool { Address }
  }
}
```
  </Tab>
  <Tab title="MCP Tool">
```
trade_get({ chain: "solana", tokenAddress: "TOKEN_ADDRESS", limit: 20 })
```
  </Tab>
</Tabs>

## 如何在 Pump.fun 上建立代幣？

ChainStream 支援透過 API 直接在 Pump.fun 上建立代幣。

<Tabs>
  <Tab title="REST API">
```bash
curl -X POST "https://api.chainstream.io/v2/dex/sol/create" \
  -H "X-API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "launchpad": "pumpfun",
    "name": "My Token",
    "symbol": "MYTKN",
    "description": "A test token",
    "privateKey": "YOUR_PRIVATE_KEY"
  }'
```

完整說明見 [DEX - Create](/zh-Hant/api-reference/endpoint/defi/dex/v2/dex-chain-create-post)。
  </Tab>
  <Tab title="MCP Tool">
```
pumpfun_create({
  name: "My Token",
  symbol: "MYTKN",
  description: "A test token"
})
```
  </Tab>
</Tabs>

<Note>
建立代幣需要已充值的 Solana 錢包。私鑰用於簽名建立交易。
</Note>

## 常見問題

<AccordionGroup>
  <Accordion title="什麼是 Pump.fun 畢業？">
    當 Pump.fun 代幣的 bonding curve 達到 100% 時，代幣會「畢業」——流動性遷移到 Raydium 池，代幣開始在公開市場交易。使用 `ranking/sol/finalStretch` 介面跟蹤接近畢業的代幣，使用 `ranking/sol/migrated` 檢視已畢業的代幣。
  </Accordion>
  <Accordion title="如何過濾機器人/MEV 成交？">
    GraphQL 的 `DEXTrades` Cube 預設帶有 `IsSuspect = false` 過濾，可自動排除疑似機器人與 MEV 成交。使用 REST API 時，請在成交響應中檢視 `isSuspect` 欄位。
  </Accordion>
  <Accordion title="能否實時跟蹤 Pump.fun 代幣？">
    可以。使用 WebSocket 訂閱獲取實時價格與成交流。配置方式見 [實時流式傳輸](/zh-Hant/guides/data-concepts/realtime-streaming)。
  </Accordion>
  <Accordion title="哪些鏈支援 Pump.fun？">
    Pump.fun 僅支援 Solana。所有 Pump.fun 相關查詢請使用 `chain: sol`（REST）或 `network: sol`（GraphQL）。
  </Accordion>
</AccordionGroup>
